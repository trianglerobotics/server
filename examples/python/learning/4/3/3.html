파이썬 반복분

Python에는 두 개의 기본 반복문 명령이 있습니다.

for 반복문

while 반복문

While 반복문

while반복문을 사용하면 조건이 참인 동안 일련의 명령문을 실행할 수 있습니다.

A 를 실행하여 출력 결과를 확인하세요

i가 6보다 작은 경우 i를 출력합니다.

          
i = 1
while i < 6:
  print(i)
  i += 1


break

break문을 사용하면 while 조건이 참이더라도 루프를 중지할 수 있습니다.

B 를 실행하여 출력 결과를 확인하세요

i가 3일 때 루프를 종료합니다.

          
i = 1
while i < 6:
  print(i)
  if i == 3:
    break
  i += 1
