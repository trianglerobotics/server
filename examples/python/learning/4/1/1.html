연산자란?

   파이썬 연산자는 데이터를 계산하거나 비교하여 조건을 만들어 내는 중요한 도구입니다. 조건은 프로그램의 논리와 흐름을 구성하는 데 사용되기 때문에 알고리즘 설계를 위해 반드시 숙지해야 하는 개념입니다.




2

산술 연산자

   산술 연산자는 데이터를 계산해서 새로운 값을 만들어 내는 데 사용합니다. 덧셈, 뺄셈, 곱셈, 나눗셈 같은 기본적인 사칙 연산부터 수를 다루는 기본적인 연산을 수행합니다.

연산자 기능
+ 덧셈을 수행한다.
- 뺄셈을 수행한다.
* 곱셈을 수행한다.
/ 나눗셈을 수행한다.
// 값을 나눈 몫을 반환한다.
% 값을 나눈 나머지를 반환한다.
** 거듭제곱을 수행한다.

할당 연산자

   할당 연산자는 데이터를 변수에 저장하는 데 사용하는 연산자입니다. 산술 연산자와 결합하여 저장된 데이터에 새로운 연산 결과를 반영하는 데 사용할 수도 있습니다.

연산자 기능
= 오른쪽의 데이터를 왼쪽 변수에 저장합니다.
+= 덧셈한 값을 저장합니다.
-= 뺄셈한 값을 저장합니다.
*= 곱한 값을 저장합니다.
/= 나눈 값을 저장합니다.
//= 값을 나눈 몫을 저장합니다.
%= 값을 나눈 나머지를 저장합니다.
**= 거듭제곱한 값을 저장합니다.




Code D

   할당 연산자를 사용해 10이 저장된 변수에 3을 더해 다시 저장해 봅시다.

Code D 를 실행하여 출력 결과를 확인하세요.
