파이썬 조건문

파이썬 조건문과 if 문

Python은 수학의 일반적인 논리적 조건을 지원합니다.

    같음: a == b
    같지 않음: a != b
    보다 작음: a < b
    보다 작거나 같음: a <= b
    보다 크다: a > b
    보다 크거나 같음: a >= b

이러한 조건은 여러 가지 방법으로 사용될 수 있지만, 가장 일반적으로는 "if 문"과 루프에서 사용됩니다.

"if 문"은 if 키워드를 사용하여 작성됩니다

A 를 실행하여 출력 결과를 확인하세요

          
a = 33
b = 200
if b > a:
  print("b is greater than a")


이 예에서 우리는 두 개의 변수 a 와 b를 사용하는데 , 이는 if 문의 일부로 사용되어 b 가 a 보다 큰지 여부를 테스트합니다 . a 는 33 이고 b 는 200 이므로 200이 33보다 크다는 것을 알고 있으므로 "b는 a보다 크다"를 화면에 인쇄합니다.

파이썬 들여쓰기

들여쓰기는 코드 줄의 시작 부분에 있는 공백을 말합니다.

다른 프로그래밍 언어에서는 코드의 들여쓰기가 가독성을 높이기 위한 것일 뿐이지만, Python에서는 들여쓰기가 매우 중요합니다.

파이썬은 들여쓰기를 사용하여 코드 블록을 나타냅니다.

*들여쓰기는 tab 키를 통하여 구현 할 수 있습니다*

B 를 실행하여 출력 결과를 확인하세요

          
if 5 > 2:
  print("Five is greater than two!")


들여쓰기를 건너뛰면 Python에서 오류가 발생합니다.

C 를 실행하여 출력 결과를 확인하세요

          
if 5 > 2:
print("Five is greater than two!")


공백의 개수는 프로그래머의 취향에 따라 결정되며, 가장 일반적인 사용은 4개이지만 적어도 1개는 있어야 합니다.

D 를 실행하여 출력 결과를 확인하세요

          
if 5 > 2:
  print("Five is greater than two!") 
if 5 > 2:
        print("Five is greater than two!")


동일한 코드 블록에서 동일한 수의 공백을 사용해야 합니다. 그렇지 않으면 Python에서 오류가 발생합니다.

E 를 실행하여 출력 결과를 확인하세요

          
if 5 > 2:
  print("Five is greater than two!")
          print("Five is greater than two!")

elif

elif 키워드 는 "이전 조건이 참이 아니면 이 조건을 시도하세요"라고 말하는 Python 방식입니다.

G 를 실행하여 출력 결과를 확인하세요

          
a = 33
b = 33
if b > a:
  print("b is greater than a")
elif a == b:
  print("a and b are equal")

이 예에서 a는 b 와 같으 므로 첫 번째 조건은 참이 아니지만 elif 조건은 참이므로 "a와 b는 같다"를 화면에 인쇄합니다.



else

else 키워드 는 이전 조건에 해당되지 않는 모든 것을 포착합니다.

H 를 실행하여 출력 결과를 확인하세요

          
a = 200
b = 33
if b > a:
  print("b is greater than a")
elif a == b:
  print("a and b are equal")
else:
  print("a is greater than b")


and

and 키워드 는 논리 연산자이며 조건문을 결합하는 데 사용됩니다.

I 를 실행하여 출력 결과를 확인하세요

a가 b 보다 큰지 그리고 c 가 a 보다 큰지 테스트합니다

          
a = 200
b = 33
c = 500
if a > b and c > a:
  print("Both conditions are True")


or

키워드는 or 논리 연산자이며 조건문을 결합하는 데 사용됩니다.

J 를 실행하여 출력 결과를 확인하세요

a가 B 보다 크거나 a가 c 보다 큰지 테스트합니다 c.

          
a = 200
b = 33
c = 500
if a > b or a > c:
  print("At least one of the conditions is True")


not

키워드는 not 논리 연산자이며 조건문의 결과를 반전하는 데 사용됩니다.

K 를 실행하여 출력 결과를 확인하세요

a 보다 b가 크지 않은지 테스트합니다 

          
a = 33
b = 200
if not a > b:
  print("a is NOT greater than b")
