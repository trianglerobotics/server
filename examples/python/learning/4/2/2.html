비교 연산자

   비교 연산자는 두 값을 비교해 그 관계를 판단할 때 사용하며, 비교 연산자를 통해 값을 비교한 식은 조건식이라고 부릅니다. ‘같다’, ‘다르다’, ‘크다’, ‘작다’와 같은 조건을 평가할 수 있으며, 결과는 참(True) 또는 거짓(False)으로 반환됩니다.


   수학에서 사용하는 등호나 부등호와 유사하다고 생각하면 이해하기 쉽습니다. 다만, 프로그래밍에서 사용하는 비교 연산 부호는 수학과 다른 경우가 있어 주의가 필요합니다.

연산자 기능
== 두 값이 같다.
!= 두 값이 다르다.
> 왼쪽 값이 크다.
>= 왼쪽 값이 크거나 같다.
< 왼쪽 값이 작다.
<= 왼쪽 값이 크거나 작다.

논리 연산자

   논리 연산자는 여러 조건을 결합해 하나의 판단 기준을 만들어 주는 기능을 합니다. 여러 가지 조건을 한 번에 처리해야 할 때 유용하게 사용합니다.

연산자 기능
and 모든 조건을 만족해야 True를 반환한다.
or 여러 조건 중 하나만 만족해도 True를 반환한다.
not 불리언(Boolean) 값을 뒤집는다.
