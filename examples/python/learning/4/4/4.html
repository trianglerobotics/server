if-elif-else문

   elif는 ‘else if’의 줄임말로, 앞의 조건식이 거짓(False)일 경우 이어서 새로운 조건을 검사한다는 의미입니다. elif는 필요한 만큼 추가할 수 있어 복잡한 조건도 하나의 코드 블록으로 처리할 수 있습니다.


if-elif-else문 형식


if-elif-else문 실행 순서
① if문 조건식이 참이면 첫 번째 결과를 출력합니다.
③if문 조건식이 거짓이고, elif문 조건식이 참이면 다음 결과를 출력합니다.
③if문과 elif문의 조건식이 모두 거짓이면, else에 해당하는 결과를 출력합니다.




Code A

   if-elif-else문을 사용해 나이가 20세 이상이면 “성인”, 20세 미만이고 13세 이상이면 “청소년”, 13세 미만은 “어린이”를 출력하는 조건문을 작성해 봅시다. (단, 조건문 실행 시 나이는 17세로 가정합니다.)
