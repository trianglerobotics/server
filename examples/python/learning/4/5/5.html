if 중첩문

   if문은 여러 개 중첩해서 사용할 수 있습니다. 입력된 데이터를 단계적으로 검사하거나, 조건을 세분화하여 각각의 경우에 맞는 결과를 내보내야 할 때 유용합니다.





Code A

   if 중첩문을 사용해 다음 조건을 만족하는 자동 대출 시스템의 알고리즘을 설계해 봅시다. (단, 조건문 실행 시 방문자는 “도서관 회원”, “연체 없음”, “A시”에 거주한다고 가정합니다.)

① 도서관 회원이어야 대출 가능합니다.
③ 연체된 도서가 없어야 대출 가능합니다.
③ 거주지가 “B시”면 7일 연장 대출 가능합니다.
Code A 를 실행하여 출력 결과를 확인하세요.




Code B

   if 중첩문을 사용해 다음 조건을 만족하는 자동 주문 시스템의 알고리즘을 설계해 봅시다. (단, 조건문 실행 시 방문자는 “차”, “라지”를 선택한다고 가정합니다.)

① 손님은 메뉴와 사이즈를 선택할 수 있습니다.
③ 메뉴는 총 두 종류로 “커피”와 “차”입니다.
③ 이외의 메뉴를 선택하면 “현재 판매하지 않습니다.”를 안내해야 합니다.
④ 음료 사이즈는 “스몰”, “미디엄”, “라지”로 구분됩니다.
⑤ 라지 사이즈 컵이 없어 현재 판매할 수 없습니다. 손님에게 “품절되었습니다.”를 안내해야 합니다.
⑥ 사이즈 선택 창에서 다른 버튼을 누르면 “사이즈를 선택해 주세요.”를 안내해야 합니다.
Code B 를 실행하여 출력 결과를 확인하세요.
