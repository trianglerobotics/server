time 라이브러리의 기능

   time 라이브러리는 시간을 다루는 기능을 제공하는 파이썬의 내장 라이브러리입니다. time을 통해 프로그램 작동 시간을 확인하거나, 프로그램 실행을 일정 시간 동안 지연시키는 등 시간과 관련된 다양한 작업을 수행할 수 있습니다. 아래에 소개하는 함수 이외에도 다양한 시간 관련 기능이 있습니다.




2

시간 확인 함수

  협정 세계시(UTC, Universal Time Coordinated)의 기준 시간인 1970년 1월 1일 0시 0분 0초를 기준으로 현재까지 경과된 시간을 초 단위로 반환합니다. 마이크로초(10⁻⁶) 단위까지 표현할 수 있습니다.



시간 확인 함수
time.time()
  • 현재 시간을 반환합니다.

3

실행 지연 함수

   프로그램 실행을 지정한 시간만큼 지연시킬 수 있습니다. 잠시 대기한 후 일정 시간 뒤에 수행해야 하는 작업이나, 일정한 간격을 두고 반복되는 작업을 처리할 때 주로 사용합니다. 밀리초(10⁻³)단위까지 입력할 수 있습니다.



실행 지연 함수
time.sleep(seconds)
  • seconds: 지연 시간을 초 단위로 입력합니다. 정수형과 실수형을 모두 입력할 수 있습니다.
