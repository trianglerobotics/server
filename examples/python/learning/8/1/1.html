라이브러리(library)란?

   라이브러리는 개발자들이 자주 사용하는 기능을 모아 언제든지 다시 사용할 수 있도록 만들어 놓은 코드의 집합체입니다. 복잡한 코드를 직접 작성하지 않고도 라이브러리를 통해 다양한 작업들을 간단하게 처리할 수 있습니다.

   이름에서 알 수 있듯이 라이브러리는 장르에 따라 책을 분류해 놓은 도서관과 같습니다. 수학 계산과 관련된 기능이 필요하다면 ‘math’ 라이브러리를, 이미지 처리를 하고 싶다면 ‘PIL(Python Image Library)’를 불러오면 됩니다.


   파이썬 내부에 기본적으로 탑재되어 있는 내장 라이브러리는 필요하다면 바로 호출할 수 있지만, 외장 라이브러리는 외부 소스를 통해 설치해야만 사용할 수 있습니다.

   자주차 SW에는 NumPy, OpenCV, Pytorch가 이미 설치되어 있어 여러분들이 언제든 호출해서 사용할 수 있습니다.

라이브러리 불러오기

   라이브러리를 이용하기 위해서는 라이브러리 호출 예약어인 ‘import’를 통해 필요한 기능이 포함된 라이브러리를 먼저 불러와야 합니다.

   import는 라이브러리의 별칭을 지정하는 키워드인 ‘as’와 함께 사용하기도 합니다. 라이브러리의 기능을 사용하려면 기능 앞에 라이브러리 이름을 작성해야 하는데, 이때 이름이 길면 입력하기 번거롭기 때문에 짧은 이름으로 별칭을 지정하는 것입니다.




라이브러리 불러오기
import라이브러리명 as 별칭
  • import: 라이브러리를 호출하는 예약어입니다.
  • as: 별칭을 지정하는 키워드입니다. 별칭이 필요하지 않다면 생략해도 됩니다.
