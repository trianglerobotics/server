인스턴스 메소드(instance method)

   인스턴스 메소드는 생성된 특정 객체를 기반으로 작동하는 함수입니다. 생성자를 통해 인스턴스 변수가 포함된 객체를 생성했다면, 인스턴스 메소드를 통해 해당 인스턴스 변수의 데이터를 확인하거나 수정할 수 있습니다.

   인스턴스 메소드를 정의할 때는 첫 번째 매개 변수로 ‘self’를 입력해야 합니다. 각각의 객체들을 독립적으로 관리한다는 의미입니다.



인스턴스 메소드 사용 예시
  • new1.change_lead("4B"): ‘객체명.메소드명’으로 인스턴스 메소드를 호출하여 실행합니다. 생성된 객체 ‘new1’의 심 종류를 4B로 바꿉니다./li>
  • new1.information(): 생성된 객체 ‘new1’의 정보를 반환하는 메소드입니다.

클래스 메소드(class method)

   클래스 메소드는 클래스 외부에서 클래스 변수를 반환하거나 관리할 수 있는 함수입니다. 굳이 클래스 내부에서 직접 변수를 찾지 않아도, 클래스 메소드만으로 클래스 변수를 확인하거나 수정할 수 있습니다.

   클래스 메소드를 정의하기 위해서는 정의에 앞서 반드시 ‘@classmethod’를 입력해야 합니다. ‘@’로 시작하는 이 문자열은 ‘데코레이터(decorator)’라고 하며, 이후 작성될 코드에 클래스 메소드 기능을 부여합니다.



클래스 메소드 기본 형식
  • @classmethod: 클래스 메소드를 정의하기 위한 데코레이터입니다.
  • def: 메소드 정의를 시작하는 예약어입니다.
  • cls: 클래스 메소드의 필수 매개 변수입니다. 클래스 변수를 참조한다는 것을 의미합니다.
  • para_n: 매개 변수입니다. 일반적인 함수의 매개 변수와 동일하게 기능합니다. 대개 클래스 외부에서 특정 클래스 변수만 호출하거나 수정하기 위해 사용합니다.


클래스 메소드 사용 예시
  • pencil.set_length(12): 클래스 메소드를 실행하여, length를 12로 바꿉니다. 클래스 메소드는 ‘클래스명.메소드명’으로 사용할 수 있습니다.
