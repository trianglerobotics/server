while 반복문

   while 반복문은 조건식이 참(True)인 동안 작업을 반복적으로 수행하는 파이썬의 반복문입니다. 반복 횟수가 명확히 정해지지 않은 상황에서 주로 사용되며, 조건식이 거짓(False)이 되면 반복을 종료합니다.

   조건식이 참이면 whiie 반복문은 멈추지 않고 계속 실행됩니다. 만약 조건식에 반복 가능한 객체(iterable)가 아니라 값이 고정된 변수를 사용하면, 조건식이 항상 참이 되어 반복문이 멈추지 않는 무한 루프에 빠질 수 있습니다.

   따라서 값이 고정된 변수를 사용할 때는 ‘=+’나 ‘=-’ 같이 산술을 포함하는 할당 연산자를 사용해 반복문이 한 번 실행된 후 값이 변할 수 있도록 만들어 주어야 합니다.


while 반복문 기본 형식
  • 조건식: while 반복문을 가능하게 하는 핵심 요소입니다. 조건식이 참(True)이면 코드를 실행하고, 거짓(False)이면 종료합니다.
  • 실행 코드: 반복 가능한 객체의 개별 데이터에 적용할 코드입니다. 반복문도 하나의 코드 블록이므로 들여쓰기를 준수해야 합니다.
