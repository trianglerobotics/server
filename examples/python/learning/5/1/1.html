반복문이란?

   반복문이란 동일한 작업을 여러 번 수행해야 할 때 사용하는 도구입니다. 범위나 조건을 설정하면 컴퓨터가 알아서 정해진 작업을 반복적으로 수행합니다.

   여러분이 매일같이 학교에 가는 일처럼 반복되는 작업을 반복문으로 표현하면 매우 간단하게 구현할 수 있습니다.




2

for 반복문

   for 반복문은 입력된 데이터 하나하나에 같은 작업을 반복적으로 적용합니다. 영어에서 전치사 for가 "~에 대해"라는 의미도 가지고 있듯, 파이썬의 for 반복문도 각각의 데이터에 대해 동일한 작업을 적용한다고 생각하면 쉽게 이해할 수 있습니다.


for 반복문 기본 형식
  • 반복문 변수: 반복문 실행 중 결과를 저장하는 변수입니다. 반복문 내부에서만 사용되지만, 원래 사용하던 변수명을 재사용하면 기존 데이터가 변경될 수 있습니다.
  • iterable: 반복 가능한 객체입니다. 시퀀스 같이 데이터가 여러 개 포함되어 있는 유형을 의미합니다.
  • 실행 코드: 반복 가능한 객체의 개별 데이터에 적용할 코드입니다. 반복문도 하나의 코드 블록이므로 들여쓰기를 준수해야 합니다.

문자열 포맷팅(formatting)
print( f"문자열{변수}" )
  • 문자열: 출력할 전체 문자열 서식입니다.
  • 변수: 문자열에 포함하고자 하는 데이터가 저장된 변수를 입력하면, 문자열과 데이터가 함께 출력됩니다.
