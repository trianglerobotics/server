파이썬 변수

변수

변수는 데이터 값을 저장하는 컨테이너입니다.

변수생성

Python에는 변수를 선언하는 명령이 없습니다.

변수는 처음으로 값을 할당하는 순간 생성됩니다.

A

x = 5 y = "John" print(x) print(y)


변수는 특정 유형 으로 선언될 필요가 없으며 , 설정된 후에도 유형을 변경할 수 있습니다.

B

x = 4 # x is of type int x = "Sally" # x is now of type str print(x)


캐스팅

변수의 데이터 유형을 지정하려면 캐스팅을 사용하면 됩니다.

C

x = str(3) # x will be '3' y = int(3) # y will be 3 z = float(3) # z will be 3.0
