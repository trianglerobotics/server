변수

변수는 데이터 값을 저장하는 바구니입니다.

변수생성

변수는 처음으로 값을 할당하는 순간 생성됩니다.

A를 실행하여 출력 결과를 확인하세요

          
x = 5
y = "John"
print(x)
print(y)


변수는 특정 유형 으로 선언될 필요가 없으며 , 설정된 후에도 유형을 변경할 수 있습니다.

B를 실행하여 출력 결과를 확인하세요

          
x = 4       # x is of type int
x = "Sally" # x is now of type str
print(x)
