변수 생성하기

   변수는 파이썬에서 데이터를 사용하기 위한 기본적인 도구입니다. 이름표가 붙은 바구니에 물건을 담는 것과 같은 역할을 합니다.


   자율주행 자동차가 센서와 카메라를 통해 주변 환경을 인식하면 데이터가 쌓입니다. 자율주행 시스템을 만들기 위해서는 이 데이터들을 활용해 최적의 알고리즘을 설계해야 합니다.


변수의 구성 요소
변수명 = 데이터
  • 변수명: 데이터를 저장할 변수의 이름입니다.
  • =(할당 연산자): 오른쪽의 데이터를 왼쪽의 변수에 저장해 주는 기호입니다. 수학의 등호처럼 양변의 값이 같다는 뜻이 아닙니다.
  • 데이터: 변수에 저장할 데이터, 즉 값입니다.

   “apple”이라는 단어를 “a”라는 변수에 저장해 봅시다.
