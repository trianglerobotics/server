출력 변수

Python print() 함수는 종종 변수를 출력하는 데 사용됩니다.

A를 실행하여 출력 결과를 확인하세요

          
x = "Python is awesome"
print(x)


함수 에서는 print() 쉼표로 구분된 여러 변수를 출력합니다.

B를 실행하여 출력 결과를 확인하세요

          
x = "Python"
y = "is"
z = "awesome"
print(x, y, z)


또한 + 연산자를 사용하여 여러 변수를 출력할 수 있습니다.

C를 실행하여 출력 결과를 확인하세요

          
x = "Python "
y = "is "
z = "awesome"
print(x + y + z)
