파이썬 - 출력 변수

출력 변수

Python print() 함수는 종종 변수를 출력하는 데 사용됩니다.

변수생성

Python에는 변수를 선언하는 명령이 없습니다.

변수는 처음으로 값을 할당하는 순간 생성됩니다.

A

x = "Python is awesome" print(x)


함수 에서는 print() 쉼표로 구분된 여러 변수를 출력합니다.

B

x = "Python" y = "is" z = "awesome" print(x, y, z)


또한 + 연산자를 사용하여 여러 변수를 출력할 수 있습니다.

C

x = "Python " y = "is " z = "awesome" print(x + y + z)
