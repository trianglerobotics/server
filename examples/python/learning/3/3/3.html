불리언 유형

   불리언(Boolean) 유형은 참(True)과 거짓(False)을 표현하는 데이터 유형입니다. 기호 논리학의 선구자인 조지 불(George Boole)의 이름을 딴 개념으로, 주로 조건문과 반복문에서 프로그램의 동작을 제어하는 데 사용됩니다.


   불리언 값은 True와 False로 표기하고, 첫 글자는 반드시 대문자로 입력해야 합니다.





Code A

   변수 rainy_day에 True 값을 부여해 봅시다.
