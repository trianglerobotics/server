특정 문자 추출하기

   문자열의 각 문자는 위치 데이터인 인덱스(index)를 가지고 있습니다. 왼쪽부터 시작하며 첫 번째 문자의 인덱스는 0입니다.


   인덱스를 활용해 원하는 문자를 하나씩 추출하는 것을 인덱싱(indexing)이라고 합니다. 추출하고자 하는 문자의 인덱스를 하나하나 정확하게 입력해야 합니다.

   슬라이싱(slicing)을 사용해 원하는 여러 개의 문자를 추출할 수도 있습니다. 슬라이싱은 콜론(:)을 사용해 범위를 지정하여 원하는 데이터만 추출하는 방법입니다.


문자열 인덱싱
data[index]
  • data: 인덱싱할 데이터입니다.
  • index: 추출할 인덱스 번호입니다.

문자열 슬라이싱
data[start:end:step]
  • data: 슬라이싱할 데이터입니다.
  • start: 슬라이싱을 시작할 문자의 인덱스입니다. 생략 시 기본값인 0으로 지정됩니다.
  • end: 슬라이싱이 종료되는 문자의 인덱스 번호에 1을 더한 값을 입력합니다. 예를 들어 종료 지점의 인덱스가 3이라면 4를 입력합니다.
  • step: 인덱스를 건너뛰는 간격입니다. 기본값은 1이며, 음수를 입력하면 역방향으로 진행됩니다.
