파이썬 숫자

Python에는 세 가지 숫자형이 있습니다.

    int
    float
    complex

숫자형 변수는 값을 할당하면 생성됩니다.

A

x = 1 # int y = 2.8 # float z = 1j # complex


Python에서 객체의 유형을 확인하려면 다음 type()함수를 사용합니다.

B

print(type(x)) print(type(y)) print(type(z))


INT

Int, 즉 정수는 소수점을 포함하지 않는 양수 또는 음수를 포함한 무한 길이의 정수입니다.

C

정수:

x = 1 y = 35656222554887711 z = -3255522 print(type(x)) print(type(y)) print(type(z))


FLOAT

부동 소수점 숫자 또는 "부동 소수점 숫자"는 소수점 하나 이상을 포함하는 양수 또는 음수 숫자입니다.

D

실수:

x = 1.10 y = 1.0 z = -35.59 print(type(x)) print(type(y)) print(type(z))
