부울은 True 또는 False 두 값 중 하나를 나타냅니다

부울 값

프로그래밍에서는 표현식이 True인지False 인지 알아야 하는 경우가 많습니다 False.

Python에서는 모든 표현식을 평가하여 True , False 두 가지 답 중 하나를 얻을 수 있습니다

두 값을 비교하면 표현식이 평가되고 Python은 부울 답을 반환합니다.

A

print(10 > 9) print(10 == 9) print(10 < 9)


if 문에서 조건을 실행하면 Python은 True or False 를 반환합니다

B

조건이 다음과 같은지 여부에 따라 메시지를 인쇄합니다

a = 200 b = 33 if b > a: print("b is greater than a") else: print("b is not greater than a")


값과 변수 평가

bool()기능을 사용하면 모든 값을 평가하고 그에 대한 대가를 받을 수 있습니다

C

문자열과 숫자를 평가합니다.

print(bool("Hello")) print(bool(15))


D

두 가지 변수를 평가합니다.

x = "Hello" y = 15 print(bool(x)) print(bool(y))
