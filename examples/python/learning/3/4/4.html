1

시퀀스(sequence)란?

   시퀀스(sequence)는 여러 데이터를 하나의 변수에 저장할 수 있는 유형입니다. 데이터들이 순차적으로 배열된 구조라고 이해할 수 있습니다.


   시퀸스에 속한 각각의 데이터는 인덱스를 가지고 있습니다. 이를 통해 특정 데이터를 추출하거나 조작할 수 있습니다. 앞서 배운 문자열도 시퀀스에 속합니다.



2

리스트(list) 유형

   리스트는 숫자, 문자열, 불리언 등 다양한 유형의 데이터를 저장할 수 있는 시퀀스입니다. 서로 다른 유형의 데이터를 한 리스트에 저장하는 것도 가능합니다. 리스트는 쉼표(,)로 구분된 데이터를 대괄호([ ])로 묶어 생성합니다.

튜플(tuple) 유형

   튜플은 리스트와 마찬가지로 하나의 변수에 여러 데이터를 저장할 수 있는 시퀀스입니다. 하지만 튜플은 수정 불가능한 목록 유형으로, 한 번 생성하면 수정할 수 없습니다. 튜플은 쉼표(,)로 구분된 데이터를 소괄호’()’로 묶어 생성합니다.

범위(range) 유형

   range는 특정 범위의 숫자(정수)를 생성하는 시퀀스입니다. 생성하고자 하는 범위의 시작 숫자와 끝 숫자를 지정하면 해당 하는 값을 순차적으로 반환합니다.

   range는 튜플과 마찬가지로 한 번 생성하면 데이터를 수정할 수 없습니다. 또한 필요한 시점에만 값을 계산하기 때문에 출력된 전체 범위를 보고 싶다면 리스트나 튜플 형태로 변환해야 합니다.


범위 생성하기
range[start, end, step]
  • start: 생성할 범위의 시작 숫자입니다.
  • end: 생성할 범위의 마지막 숫자에 1을 더한 값입니다. 10까지 생성하고 싶다면, 11을 입력하면 됩니다.
  • step: 범위 생성 시 규칙입니다. 기본 값은 1이며 건너뛰고 싶은 만큼 숫자를 입력하면 됩니다.
