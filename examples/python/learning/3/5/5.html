파이썬 리스트

리스트

리스트는 여러 항목을 하나의 변수에 저장하는 데 사용됩니다

리스트는 파이썬에 내장된 4가지 데이터 유형 중 하나로, 데이터 컬렉션을 저장하는 데 사용됩니다. 나머지 3가지는 튜플(Tuple ), 셋(Set ), 딕셔너리(Dictionary) 이며 , 각각 품질과 사용법이 다릅니다.

리스트는 대괄호를 사용하여 생성됩니다.

A 를 실행하여 출력 결과를 확인하세요

          
thislist = ["apple", "banana", "cherry"]
print(thislist)


리스트 항목

리스트 항목은 정렬되고 변경 가능하며 중복된 값을 허용합니다.

B 를 실행하여 출력 결과를 확인하세요

          
thislist = ["apple", "banana", "cherry", "apple", "cherry"]
print(thislist)


리스트 길이

목록에 항목이 몇 개 있는지 확인하려면 다음 len()함수를 사용하세요.

C 를 실행하여 출력 결과를 확인하세요

          
thislist = ["apple", "banana", "cherry"]
print(len(thislist))


리스트 항목 - 데이터 유형

리스트 항목은 모든 데이터 유형이 될 수 있습니다.

D 를 실행하여 출력 결과를 확인하세요

          
list1 = ["apple", "banana", "cherry"]
list2 = [1, 5, 7, 9, 3]
list3 = [True, False, False]
