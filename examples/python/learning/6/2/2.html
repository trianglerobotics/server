모듈에서 특정 항목만 가져오기

모듈의 일부만 가져와 사용하는 방식입니다.

필요한 기능만 가져오기 때문에 코드가 간결해집니다.

A 를 실행하여 출력 결과를 확인하세요

          
from math import sqrt
print(sqrt(16))  # sqrt 함수만 사용 가능


B 를 실행하여 출력 결과를 확인하세요

          
from datetime import datetime, timedelta

# 현재 시간 출력
now = datetime.now()
print(f"현재 시간: {now}")

# 하루 더한 날짜 계산
tomorrow = now + timedelta(days=1)
print(f"내일 날짜: {tomorrow}")


C 를 실행하여 출력 결과를 확인하세요

          
from os import getcwd, listdir

# 현재 디렉토리 가져오기
current_dir = getcwd()
print(f"현재 디렉토리: {current_dir}")

# 디렉토리 목록 가져오기
files = listdir(current_dir)
print(f"디렉토리 파일 목록: {files}")
