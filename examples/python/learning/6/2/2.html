1

함수명(function name)

   ‘함수명(function name)’은 함수를 호출하기 위해 사용하는 고유한 이름입니다. 보통 함수가 수행하는 작업을 직관적으로 반영한 이름이 사용됩니다.




2

매개 변수(parameter)

   ‘매개 변수(parameter)’는 값을 입력 받을 위치를 표시해 주는 표지입니다. 함수를 사용하기 위해서는 지정된 위치에 필요한 값을 입력해야 합니다. 이때 매개 변수는 입력 값이 어떤 위치에 들어가야 하는지 알려주는 역할을 합니다.




3

인수(argument)

   ‘인수(argument)’는 실제로 함수에 입력되는 값입니다. 매개 변수에 대응하는 위치에 적절한 인수를 입력하면, 함수 내부의 코드를 거쳐 일정한 결과를 반환합니다.




4

함수 본문(fuction body)

   ‘함수 본문(function body)’은 실제 작업을 수행하는 함수 내부의 코드입니다. 내장 함수의 본문은 파이썬 내부에 이미 구현되어 있어 직접 볼 수는 없지만, 본문에 작성된 코드에 의해 특정 작업이 수행됩니다.




5

반환 값(return value)

   ‘반환 값(return value)’은 함수 내부에서 작업을 마친 후 사용자에게 돌려주는 결괏값입니다. 반환 값은 다른 변수에 저장하거나 이후 코드에 바로 적용할 수 있습니다.
