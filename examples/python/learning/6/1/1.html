파이썬 import

전체모듈 가져오기

모듈 전체를 가져와 사용할 때 사용하는 방식입니다.

모듈 이름을 명시적으로 작성해야 하므로 코드 가독성이 좋지만, 길이가 길어질 수 있습니다.

A 를 실행하여 출력 결과를 확인하세요

          
import math
print(math.sqrt(16))  # math 모듈의 sqrt 함수 사용


B 를 실행하여 출력 결과를 확인하세요

          
import datetime

# 현재 날짜와 시간 가져오기
now = datetime.datetime.now()
print(f"현재 시간: {now}")

# 특정 날짜 생성
specific_date = datetime.datetime(2023, 11, 18)
print(f"특정 날짜: {specific_date}")


C 를 실행하여 출력 결과를 확인하세요

          
import os

# 현재 작업 디렉토리 확인
current_directory = os.getcwd()
print(f"현재 디렉토리: {current_directory}")

# 디렉토리 목록 출력
print(f"디렉토리 목록: {os.listdir(current_directory)}")
