1

함수의 기본 구조

   기본적으로 제공되는 내장 함수 이외에도 사용자가 직접 함수를 정의하여 사용할 수 있습니다. 함수를 정의해서 사용하면 반복되는 작업도 간단하게 처리할 수 있고, 코드의 가독성을 높일 수 있습니다.

   함수를 정의하기 위해서는 ‘def’를 사용합니다. def는 define의 약자로 파이썬에서 함수를 정의할 때 사용하는 예약어입니다. 가장 기본적인 함수는 ‘def’와 ‘함수 본문’만으로 이루어진 구조입니다.


함수의 기본 구조 예시
  • def: 함수 정의를 시작하는 예약어입니다.
  • start_jajucha: 함수명입니다. 수행하는 작업을 직관적으로 파악할 수 있는 이름을 사용합니다.
  • print(···): 함수 본문입니다. 함수 호출 시 실행됩니다.


2

매개 변수를 포함하는 함수 정의

   함수명 오른쪽에 있는 괄호에 매개 변수를 넣어 값을 입력 받을 수 있습니다. 매개 변수에는 숫자형부터 문자형, 시퀀스까지 다양한 데이터 유형을 모두 입력할 수 있고, 이를 통해 다양한 입력값을 받아 여러 상황에 함수를 적용할 수 있습니다.


매개 변수를 포함한 함수 예시
  • def: 함수 정의를 시작하는 예약어입니다.
  • add_numbers: 함수명입니다.
  • num1, num2: 매개 변수입니다. 입력값이 매개 변수를 통해 함수 내부로 전달됩니다.
  • print(···): 함수 본문입니다. 함수 호출 시 실행됩니다.

   함수가 작동하여 나온 결과를 반환하여 이후 루틴에 활용할 수 있습니다. 변수에 저장해서 사용하거나 조건문이나 반복문에 필요한 값으로도 사용할 수 있습니다.


반환 값을 포함한 함수 예시
  • def: 함수 정의를 시작하는 예약어입니다.
  • check_even: 함수명입니다.
  • number: 매개 변수입니다.
  • return···: 반환 값으로 구성된 함수 본문입니다.
  • if···: 함수의 반환 값으로 작동하는 조건문입니다.
