별칭을 사용하여 가져오기

모듈 이름이 길거나 반복되는 경우, 별칭을 지정하여 사용할 수 있습니다.

주로 데이터 과학 라이브러리(Pandas, NumPy 등)에서 많이 사용됩니다.

A 를 실행하여 출력 결과를 확인하세요

          
import numpy as np
array = np.array([1, 2, 3])  # numpy의 별칭 'np'로 사용


B 를 실행하여 출력 결과를 확인하세요

          
import matplotlib.pyplot as plt

# 데이터
x = [1, 2, 3, 4]
y = [2, 4, 6, 8]

# 그래프 그리기
plt.plot(x, y)
plt.title('Line Plot Example')
plt.xlabel('X-axis')
plt.ylabel('Y-axis')
plt.show()


C 를 실행하여 출력 결과를 확인하세요

          
import torch as th

# GPU 사용 가능한지 확인
device = th.device('cuda' if th.cuda.is_available() else 'cpu')
print(f"사용 중인 디바이스: {device}")

# GPU에서 텐서 생성
gpu_tensor = th.tensor([1.0, 2.0, 3.0], device=device)
print(f"GPU 텐서:\n{gpu_tensor}")
