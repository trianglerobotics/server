조향모터 제어하기

자주차 모듈 가져오기

자주차를 제어하기 위한 함수들의 모음집인 jajucha2 라이브러리를 가져옵니다

A 를 실행하여 자주차 라이브러리를 가져옵니다

자주차 라이브러리는 인공지능 기능을 포함 하고 있어 로딩하는데 2-3초 정도 소요됩니다

          
import jajucha2


모터 제어하기

자주차를 모터를 실질적으로 제어합니다

set_motor(left,right,speed) 순서입니다

left,right = ±10 범위입니다

speed = ±30 범위입니다

B

정지상태에서 왼쪽으로 스티어링을 조향합니다

          
jajucha2.control.set_motor(-10,-10,0)


C

정지상태에서 오른쪽으로 스티어링을 조향합니다

          
jajucha2.control.set_motor(10,10,0)


D

1초 간격으로 좌우로 스티어링을 조작합니다

          
import time #시간 관련 라이브러리를 가져옵니다

while(True): 
    jajucha2.control.set_motor(10,10,0) 
    time.sleep(1) 
    jajucha2.control.set_motor(-10,-10,0)
    time.sleep(1)
