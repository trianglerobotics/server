모터 제어하기

자주차 모터 제어 함수
jajucha2.control.set_motor(left,right,speed)
  • left : 왼쪽 모터의 조향각으로 ±10 범위입니다
  • right : 왼쪽 모터의 조향각으로 ±10 범위입니다
  • speed : 자주차의 속도로 ±30 범위입니다


Code B

  Code B를 실행하여 정지상태에서 왼쪽으로 스티어링을 조향합니다



Code C

  Code C를 실행하여 정지상태에서 오른쪽으로 스티어링을 조향합니다



Code D

  Code D를 실행하여 1초 간격으로 좌우로 스티어링을 조작합니다
