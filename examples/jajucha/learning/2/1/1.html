RGB 이미지 받기, 화면에 표시하기

자주차 모듈 가져오기

자주차를 제어하기 위한 함수들의 모음집인 jajucha2 라이브러리를 가져옵니다

A 를 실행하여 자주차 라이브러리를 가져옵니다

자주차 라이브러리는 인공지능 기능을 포함 하고 있어 로딩하는데 2-3초 정도 소요됩니다

          
import jajucha2


이미지 받아오기

B를 실행합니다

왼쪽,가운데,오른쪽 이미지를 받아와서 각각의 변수에 저장합니다

          
left = jajucha2.camera.get_image('left')
center = jajucha2.camera.get_image('center')
right = jajucha2.camera.get_image('right')


left = jajucha2.camera.get_image('left') -> 왼쪽 카메라의 이미지를 받아옵니다

center = jajucha2.camera.get_image('center') -> 중간 카메라의 이미지를 받아옵니다

right = jajucha2.camera.get_image('right') -> 오른쪽 카메라의 이미지를 받아옵니다

이미지 표시하기

C를 실행하기 전에 우측 상단의 를 눌러 이미지 뷰어를 활성화 홥니다

C 를 실행합니다

받아온 이미지를 이미지 뷰어로 전송합니다

          
jajucha2.camera.show_image(left,'left')
jajucha2.camera.show_image(center,'center')
jajucha2.camera.show_image(right,'right')


jajucha2.camera.show_image(left,'left') -> 왼쪽이미지를 'left' 창으로 전송합니다

jajucha2.camera.show_image(center,'center') -> 중간 'center' 창으로 전송합니다

jajucha2.camera.show_image(right,'right') -> 오른쪽 'right' 창으로 전송합니다

D를 실행합니다

연속적으로 이미지를 전송합니다

          
import jajucha2

while True:
    center = jajucha2.camera.get_image('center')
    jajucha2.camera.show_image(center,'center')
