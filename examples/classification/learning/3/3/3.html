불러온 모델 테스트하기

CELL-1

STEP-1 : NAME 에 해당하는 이름은 전 과정에서 유저가 설정했던 모델의 이름입니다 이름을 모를 경우 Database -> Model Manager 에서 확인 가능합니다



CELL-2

STEP-1 : CELL-2 에서는 카메라의 이미지와 모델을 가져옵니다

*TIP: 평가모드로 전환하지 않으면 추후에 그레디언트를 계산하기 때문에 연산이 느려지게 됩니다



CELL-3

STEP-1 : 카메라에서 받을 이미지를 모델에 넣기 전 이미지의 크기와 모델에 넣어야 할 크기를 출력합니다

이미지의 크기는 (480, 640, 3)

모델에 넣어야 할 크기는 [1, 3, 224, 224] 라는 것을 알 수 있습니다



CELL-4

STEP-1 : CELL-4 에서는 모델에 넣어야 할 데이터의 크기를 출력합니다. 자주차 내부 함수를 이용해서 모델의 이름을 넣으면 추후에 모델에 넣어야 할 데이터의 크기를 알 수 있습니다



CELL-5

STEP-1 : CELL-5 모델에 실질적으로 임의의 데이터를 넣어서 결과를 출력합니다. 모델에 넣어야 할 데이터의 크기를 알고 있기 때문에 데이터를 생성하고 모델에 넣어서 결과를 출력합니다

*TIP: 모델이 GPU에 있기 때문에 넣으줄 값도 GPU에 위치 시켜야 합니다 device='cuda' 를 통해서 값을 GPU에서 생성합니다
