그리드 갯수 조작하기

jajucha2.camera.gridFront() 함수에서 선을 검출하는 정확도를 높이기 위해 그리드 갯수를 열은 10개 , 행은 4개로 조작합니다

코드를 실행하기 전 을 클릭하여 이미지 뷰어를 활성화 하세요

Code A 위에 cols = 10, rows = 4로 수정한 gridFront 코드를 작성합니다
정답 보기/숨기기 
            
(V,L,R), grid = jajucha2.camera.gridFront(frame,cols=10,rows=4)
