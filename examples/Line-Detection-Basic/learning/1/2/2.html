자주차 라이브러리 가져오기

자주차를 제어하기 위한 함수들의 모음집인 jajucha2 라이브러리를 가져옵니다

자주차 라이브러리 이름
jajucha2
  • jajucha2 : 자주차에 필요한 함수를 담고 있는 라이브러리 입니다

Code A

  Code A 에 자주차 라이브러리를 임포트하는 코드를 작성후 실행하세요

Code A 정답 보기/숨기기 
            
import jajucha2    

2

자주차에서 이미지 가져오기

자주차의 함수를 이용하여 RGB 함수를 받아와 변수에 저장합니다

자주차 이미지 가져오기 함수
jajucha2.camera.get_image(window_name='center')
  • location : 이미지를 가져올 카메라 위치를 지정합니다. 'left','center','right'

Code B

  Code B 에 자주차 'center' 카메라에서 RGB 이미지를 받아와서 'frame'변수에 저장하는 코드를 작성,실행하세요

Code B 정답 보기/숨기기 
            
frame = jajucha2.camera.get_image('center')    

3

이미지 표시하기

자주차 이미지 가져오기 함수
jajucha2.camera.show_image(image,window_name='center',quality=20)
  • image : 표시할 이미지를 지정합니다. numpy.ndarray 타입
  • location : 이미지를 표시할 위치를 지정합니다. 'left','center','right','depth',lidar'
  • quality : 이미지의 퀄리티를 지정합니다. 0~100 사이의 정수

Code C

  Code C 에 'frame' 이미지를 자주차 'center' 이미지뷰어에 퀄리티 '90'으로 표시합니다

코드를 실행하기 전 을 클릭하여 이미지 뷰어를 활성화 하세요

Code C 정답 보기/숨기기 
            
frame = jajucha2.camera.show_image(frame,'center',quality=90)
