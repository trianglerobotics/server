blurred 이미지에서 Edge 검출하기

opencv Canny edge detection 함수
cv2.Canny(image, threshold1, threshold2, edges=None, apertureSize=None, L2gradient=None)
  • image : 입력 이미지
  • threshold1 : 첫 번째 임계값
  • threshold2 : 두 번째 임계값
  • edges : 출력 이미지
  • apertureSize : 소벨 연산자의 크기
  • L2gradient : 그레디언트 크기를 계산할 때 사용할 방법
