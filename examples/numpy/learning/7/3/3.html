배열 균등 분할

   균등 분할 함수를 사용하여 수직 방향이나 수평 방향으로 배열을 균등하게 분할할 수 있습니다. 원본 배열은 그대로 유지하면서, 분할한 배열들을 새롭게 생성합니다.

   이때 배열이 분할하고자 하는 수로 균등하게 나누어 떨어져야 합니다. 정확하게 나누어 떨어지지 않으면 오류가 발생합니다.



   수직 분할 함수는 수직 결합 함수의 반대 방향으로 배열을 분할합니다.



   수평 분할 함수도 마찬가지로 수평 결합 함수의 반대 방향으로 배열을 분할합니다.



수직 분할 함수
np.vsplit(array, sections)
  • array: 분할하고자 하는 원본 배열입니다.
  • sections: 분할 조각 개수를 지정합니다.


수평 분할 함수
np.hsplit(array, sections)
  • array: 분할하고자 하는 원본 배열입니다.
  • sections: 분할 조각 개수를 지정합니다.

배열 지정 분할

   지정 분할 함수를 사용하면 분할할 조각 개수와 축 방향을 지정하여 배열을 분할할 수 있습니다.

   만약 기준 축의 크기가 지정한 분할 개수로 정확하게 나누어 떨어지지 않는다면, 앞쪽 배열에 더 큰 조각을 할당하여 최대한 균등하게 배분합니다.




배열 분할 함수
np.array_split(array, sections, axis=n)
  • array_n: 분할하고자 하는 원본 배열입니다.
  • sections: 분할 조각 개수를 지정합니다.
  • axis=n: 2차원 이상의 배열에서 분할 기준 축을 지정합니다. 기본값은 수직 방향 분할입니다.
