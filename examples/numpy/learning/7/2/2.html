수직 방향 결합

   수직 결합 함수를 사용해 여러 배열을 수직 방향으로 결합할 수 있습니다. 원본 배열들은 그대로 유지하면서, 결합한 배열을 새롭게 생성합니다.

   이때 결합하고자 하는 배열들의 행 축의 크기, 즉 열 개수가 동일해야 합니다.




수직 결합 함수
np.vstack(array_1, array_2, ..., array_n)
  • array_n: 수직 방향으로 결합하고자 하는 배열을 모두 입력합니다.




Code A

   수직 결합 함수를 사용해 array1에 저장된 3 X 3 크기의 배열과 array2에 저장된 2 X 3 크기의 배열을 수직 방향으로 결합해 봅시다.

Code A 를 실행하여 출력 결과를 확인하세요.

수평 방향 결합

   수평 결합 함수를 사용해 여러 배열을 수평 방향으로 결합할 수 있습니다. 원본 배열들은 그대로 유지하면서, 결합한 배열을 새롭게 생성합니다.

   이때 결합하고자 하는 배열들의 열 축의 크기, 즉 행 개수가 동일해야 합니다.




수평 결합 함수
np.hstack(array_1, array_2, ..., array_n)
  • array_n: 수직 방향으로 결합하고자 하는 배열을 모두 입력합니다.




Code B

   수평 결합 함수를 사용해 array3에 저장된 3 X 3 크기의 배열과 array4에 저장된 3 X 4 크기의 배열을 수직 방향으로 결합해 봅시다.
