배열 형태 바꾸기

   형태 바꾸기 함수는 배열의 형태를 바꾸어 새로운 배열로 만들어 주는 함수입니다. 원본 배열은 그대로 유지하면서, 형태를 바꾼 새로운 배열을 생성합니다.

   형태를 바꿀 때는 반드시 변경할 배열의 요소 개수가 원본 배열의 요소 개수와 같아야 합니다. 변경할 배열의 요소 개수가 부족하거나 더 많다면 오류가 발생합니다.

   변경할 배열의 형태는 튜플 형식으로 입력합니다. 하나의 축 크기를 -1로 입력하면, 나머지 축 크기와 원본 배열의 전체 요소 개수를 고려하여 적합한 축 크기를 자동으로 지정합니다.




형태 바꾸기 함수
np.reshape(array, new_shape)
  • array: 형태를 바꿀 원본 배열입니다.
  • new_shape: 원하는 형태를 튜플 형식으로 입력합니다.




Code A

   형태 바꾸기 함수를 사용해 array1에 저장된 4 X 5 크기의 배열을 행 크기가 10인 2차원 배열로 바꾸어 봅시다.

평탄화 메소드
arrayflatten()
  • array: 평탄화할 원본 배열입니다.
