넘파이 기본 개념

넘파이는 Numerical Python의 약자로, Python에서 과학적 계산을 위해 사용되는 수치 계산 라이브러리입니다. 주로 고성능 다차원 배열(ndarray)을 지원하며, 배열을 효율적으로 처리할 수 있는 다양한 함수와 기능을 제공합니다.

넘파이 ndarray

넘파이는 다차원 배열을 지원하며, 1D, 2D, 3D 이상의 배열을 처리할 수 있습니다.

배열은 같은 데이터 타입을 가지며, 연속적인 메모리 공간에 저장되어 효율적인 계산이 가능합니다.

numpy.ndarray(shape, dtype=float, buffer=None, offset=0, strides=None, order=None)

shape: 배열의 차원 크기

dtype: 배열의 데이터 타입

buffer: 초기 데이터를 제공하는 버퍼

offset: 버퍼에서 데이터를 읽기

strides: 각 차원의 메모리 간격

order: 배열의 메모리 저장 순서

넘파이 배열: [1 2 3 4 5]
합계: 15

정답 보기/숨기기 
          
# 넘파이 임포트
import numpy as np

# 기본적인 배열 생성
arr = np.array([1, 2, 3, 4, 5])
print("넘파이 배열:", arr)

# 배열의 합 계산
print("합계:", np.sum(arr))

# 배열 요소 제곱
print("제곱된 배열:", arr ** 2)

넘파이는 내부적으로 배열 데이터를 연속된 메모리 블록에 저장해 성능이 훨씬 빠릅니다.

import numpy as np
import time

# Python 리스트와 넘파이 배열 생성
list_data = list(range(1, 1000001))
np_data = np.array(list_data)

# Python 리스트의 연산
start = time.time()
list_result = [x * 2 for x in list_data]
print("리스트 연산 시간:", time.time() - start)

# 넘파이 배열의 연산
start = time.time()
np_result = np_data * 2
print("넘파이 연산 시간:", time.time() - start)
