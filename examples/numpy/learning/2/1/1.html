넘파이 배열 생성

배열 생성 방법

넘파이 배열은 np.array()를 사용하여 생성할 수 있습니다.

0으로 초기화된 배열, 1로 초기화된 배열, 연속적인 값으로 채운 배열 등을 생성할 수 있는 다양한 함수가 제공됩니다.

리스트로부터 생성: np.array

0으로 초기화: np.zeros

1로 초기화: np.ones

연속된 값: np.arange

선형적으로 분할: np.linspace

A에 있는 빈칸을 채운 후 A 를 실행하여 아래 결과를 확인하세요

리스트로부터 생성된 배열: [1 2 3 4]
0으로 초기화된 배열: [[0. 0. 0.] [0. 0. 0.]]
1로 초기화된 배열: [[1. 1. 1.] [1. 1. 1.] [1. 1. 1.]]
연속된 값 배열: [0 2 4 6 8]
선형적으로 분할된 배열: [0. 0.25 0.5 0.75 1. ]
정답 보기/숨기기 
          
import numpy as np

# 1. 리스트로부터 배열 생성
arr1 = np.array([1, 2, 3, 4])
print("리스트로부터 생성된 배열:", arr1)

# 2. 0으로 초기화된 배열 생성
arr2 = np.zeros((2, 3))
print("0으로 초기화된 배열:\n", arr2)

# 3. 1로 초기화된 배열 생성
arr3 = np.ones((3, 3))
print("1로 초기화된 배열:\n", arr3)

# 4. 연속된 값으로 채운 배열 생성
arr4 = np.arange(0, 10, 2)  # 0부터 10까지 2씩 증가
print("연속된 값 배열:", arr4)

# 5. 선형적으로 분할된 값으로 채운 배열 생성
arr5 = np.linspace(0, 1, 5)  # 0부터 1까지 5개의 값
print("선형적으로 분할된 배열:", arr5)
