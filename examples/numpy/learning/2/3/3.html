차원(dimension) 확인 속성

   넘파이는 배열의 차원을 확인할 수 있는 속성(attribute)인 ‘ndim’을 제공합니다. ndim은 ‘number of dimension’의 약자로 차원의 개수를 의미합니다.


ndim 속성
array.ndim
  • array: 차원을 확인하고자 하는 넘파이 배열 데이터입니다.




Code A

   ndim 속성을 사용해 array1 ,array2, array3에 저장된 넘파이 배열의 차원의 개수를 확인해 봅시다.

데이터 개수 확인 속성

   ‘size'는 배열 안에 든 전체 데이터 요소의 개수를 나타내는 속성입니다. 전체 데이터 요소의 수는 결국 축의 크기를 모두 곱한 것과 같습니다.



size 속성
array.size
  • array: 데이터 개수를 확인하고자 하는 넘파이 배열 데이터입니다.




Code C

   size 속성을 사용해 변수 array1 ,array2, array3에 저장된 넘파이 배열의 데이터 개수를 확인해 봅시다.

Code C 를 실행하여 출력 결과를 확인하세요.

dtype 속성
