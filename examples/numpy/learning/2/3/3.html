넘파이 배열 생성

데이터 타입 설정 및 변환

넘파이 배열은 모든 요소가 동일한 데이터 타입을 가집니다.

배열 생성 시 dtype을 명시하거나, 생성 후 astype()으로 타입을 변경할 수 있습니다.

배열 생성 시 데이터 타입을 설정 합니다

A 를 실행하여 결과를 확인하세요

          
import numpy as np

# 1. 배열 생성 시 데이터 타입 설정
arr1 = np.array([1, 2, 3], dtype=np.float32)
print("float32 타입 배열:", arr1)
print("데이터 타입:", arr1.dtype)


데이터 타입을 변환 합니다

B 를 실행하여 결과를 확인하세요

          
import numpy as np
# 2. 데이터 타입 변환
arr2 = np.array([1.1, 2.2, 3.3])
print("기존 배열 데이터 타입:", arr2.dtype)


int로 변환 합니다

C 를 실행하여 결과를 확인하세요

          
import numpy as np
  
# 3. int로 변환
arr2_int = arr2.astype(np.int32)
print("int32 타입으로 변환된 배열:", arr2_int)
print("변환 후 데이터 타입:", arr2_int.dtype)
