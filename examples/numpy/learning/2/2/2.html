난수 배열 생성

np.random 모듈을 사용하여 다양한 난수 생성이 가능합니다.

균등분포, 정규분포, 임의의 정수, 다차원 배열 형태의 난수 등을 생성할 수 있습니다.

0과 1 사이 균등분포 난수: np.random.rand

정규분포 난수: np.random.randn

의의 정수 배열: np.random.randint

섞인 배열: np.random.shuffle

A에 있는 빈칸을 채운 후 A 를 실행하여 아래 결과를 확인하세요

0과 1 사이 균등분포 난수: [[0.7030073 0.1337954 0.29474663] [0.89852511 0.49183264 0.92508197]]
정규분포 난수: [[ 0.97231267 -1.38079247 1.09289348] [-0.96690069 -0.6182254 1.54476182] [ 1.90540084 0.10289803 -1.47188307]]
임의의 정수 배열: [[2 2] [5 5]]
섞인 배열: [4 3 5 2 1]
정답 보기/숨기기 
          
import numpy as np

# 1. 균등분포로 난수 생성 (0과 1 사이)
rand_uniform = np.random.rand(2, 3)
print("0과 1 사이 균등분포 난수:\n", rand_uniform)

# 2. 정규분포로 난수 생성 (평균 0, 표준편차 1)
rand_normal = np.random.randn(3, 3)
print("정규분포 난수:\n", rand_normal)

# 3. 임의의 정수 생성
rand_integers = np.random.randint(1, 10, size=(2, 2))  # 1 이상 10 미만 정수
print("임의의 정수 배열:\n", rand_integers)

# 4. 배열 섞기
arr = np.array([1, 2, 3, 4, 5])
np.random.shuffle(arr)
print("섞인 배열:", arr)
