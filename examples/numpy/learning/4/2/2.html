배열 결합과 분할

배열 결합과 분할은 여러 개의 배열을 하나로 합치거나, 큰 배열을 여러 개로 나누는 작업입니다. 넘파이는 이를 위해 다양한 함수와 기능을 제공합니다.

배열 결합과 분할 - 1 : 배열의 결합

배열의 결합은 여러 개의 배열을 하나로 합치는 작업입니다

배열을 수직으로 쌓습니다 : np.vstack

배열을 수평으로 쌓습니다 :np.hstack

A를 실행하여 결과를 출력하세요

          
import numpy as np

# 두 배열 생성
arr1 = np.array([[1, 2, 3]])
arr2 = np.array([[4, 5, 6]])

# 수직으로 쌓기
result_v = np.vstack((arr1, arr2))
result_h = np.hstack((arr1, arr2))

print("수직으로 쌓은 결과:")
print(result_v)

print("수평으로 쌓은 결과:")
print(result_h)


배열 결합과 분할 - 2 : 배열의 분할

배열의 **분할(Splitting)**은 배열을 여러 조각으로 나누는 작업입니다.

numpy.split(ary, indices_or_sections, axis=0)

ary: 입력 배열

indices_or_sections 분할할 부분의 개수

axis 분할할 축

B

          
import numpy as np

# 1차원 배열
arr = np.array([1, 2, 3, 4, 5, 6])
split_arr = np.split(arr, 3)  # 배열을 3개의 부분으로 나눔
print(split_arr)

# 2차원 배열
arr2d = np.array([[1, 2, 3], [4, 5, 6]])
split_arr2d = np.split(arr2d, 3, axis=1)  # 열 기준으로 3개의 부분으로 나눔
print(split_arr2d)
