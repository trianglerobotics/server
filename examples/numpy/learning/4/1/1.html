배열의 모양과 구조 조작

배열의 차원 확인 및 변경

배열의 차원과 모양은 데이터의 구조를 이해하는 데 필수적입니다

배열의 차원 확인은 배열의 차원을 파악하는 과정입니다

배열의 차원 변경은 데이터를 재구조화합니다

배열의 차원 확인 및 변경 -1 : 차원 확인

배열의 차원(Dimension)이란 배열이 가진 데이터 구조의 레벨을 의미합니다

넘파이에서는 ndarray 객체를 사용하여 다차원 배열을 나타냅니다

A를 실행하여 결과를 출력하세요

          
import numpy as np

# 0차원 배열
arr0 = np.array(42)
print("0차원 배열:", arr0)
print("차원 수:", arr0.ndim)
print("모양(shape):", arr0.shape)

# 1차원 배열
arr1 = np.array([1, 2, 3, 4])
print("\n1차원 배열:", arr1)
print("차원 수:", arr1.ndim)
print("모양(shape):", arr1.shape)

# 2차원 배열
arr2 = np.array([[1, 2], [3, 4], [5, 6]])
print("\n2차원 배열:\n", arr2)
print("차원 수:", arr2.ndim)
print("모양(shape):", arr2.shape)

# 3차원 배열
arr3 = np.array([[[1, 2], [3, 4]], [[5, 6], [7, 8]]])
print("\n3차원 배열:\n", arr3)
print("차원 수:", arr3.ndim)
print("모양(shape):", arr3.shape)


배열의 차원 확인 및 변경 -2 : 차원 변경

배열의 차원을 변경하면 데이터의 **구조(Shape)**를 바꿀 수 있습니다.

넘파이는 차원 변경 함수를 제공하여 배열을 원하는 모양으로 쉽게 변환할 수 있습니다.

데이터의 순서는 변경되지 않으며, 총 요소 개수는 변경 전후 동일해야 합니다.

B를 실행하여 결과를 출력하세요

          
import numpy as np

# 원본 배열
arr = np.array([[1, 2, 3], [4, 5, 6]])
print("원본 배열:\n", arr)
print("차원 수:", arr.ndim)
print("모양(shape):", arr.shape)

# 1. reshape()로 차원 변경
reshaped = arr.reshape(3, 2)
print("\nreshape()로 변경된 배열:\n", reshaped)
print("변경된 모양(shape):", reshaped.shape)

# 2. -1을 사용해 자동 계산
reshaped_auto = arr.reshape(-1, 3)  # 열 개수는 3, 행은 자동 계산
print("\n-1을 사용한 배열:\n", reshaped_auto)
print("변경된 모양(shape):", reshaped_auto.shape)

# 3. ravel()로 평탄화
raveled = arr.ravel()
print("\nravel()로 평탄화된 배열:", raveled)

# 4. flatten()으로 평탄화
flattened = arr.flatten()
print("\nflatten()으로 평탄화된 배열:", flattened)

# 5. resize()로 원본 변경
arr.resize(3, 2)
print("\nresize()로 변경된 원본 배열:\n", arr)
