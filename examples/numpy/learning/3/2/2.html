넘파이 배열의 기본 연산

브로드캐스팅(Broadcasting) 개념 이해

브로드캐스팅은 크기가 다른 배열 간 연산을 가능하게 해주는 넘파이의 기능입니다. 작은 배열이 큰 배열의 크기에 맞게 확장되어 연산이 수행됩니다. 이 기능은 코드의 간결성과 성능을 크게 향상시킵니다.

브로드캐스팅 개념 이해 - 1 : 1D 배열과 스칼라 간 브로드캐스팅

배열과 스칼라 간 연산은 스칼라 값을 배열의 크기에 맞게 확장한 후, 배열의 각 요소에 대해 동일한 연산을 수행합니다.

예를 들어, arr + 10의 경우, 10이 배열의 모든 요소에 "확장"되어 각각 더해집니다.

A

          
import numpy as np
# 1D 배열과 스칼라 간 브로드캐스팅
arr = np.array([1, 2, 3])
print("배열 + 10:", arr + 10)


브로드캐스팅 개념 이해 - 2 : 1D 배열과 2D 배열 간 브로드캐스팅

1D 배열 arr2가 2D 배열 arr1의 모양에 맞게 확장됩니다. (가상의 확장)

확장된 1D 배열의 각 요소가 2D 배열의 해당 차원 요소에 연산됩니다.

B

          
import numpy as np
# 1D 배열과 2D 배열 간 브로드캐스팅
arr1 = np.array([[1, 2, 3], [4, 5, 6]])
arr2 = np.array([10, 20, 30])
print("2D 배열 + 1D 배열:\n", arr1 + arr2)


브로드캐스팅 개념 이해 - 3 : 다차원 배열 브로드캐스팅

작은 배열(여기서는 arr3 또는 arr4)이 큰 배열의 차원에 맞게 가상 확장됩니다.

확장된 배열을 기준으로 각 요소별(Element-wise) 연산이 수행됩니다.

C

          
import numpy as np
# 다차원 배열 브로드캐스팅
arr3 = np.array([[1], [2], [3]])
arr4 = np.array([10, 20, 30])
print("2D 배열과 1D 배열 브로드캐스팅:\n", arr3 + arr4)
