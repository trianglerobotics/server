빈 배열 생성하기

   자율주행은 다양한 센서를 통해 얻은 데이터를 실시간으로 반영합니다. 그런데 데이터를 반영할 때마다 배열을 생성한 후 데이터를 처리하면 많은 시간이 소요됩니다. 그래서 이때 빈 배열이나 초기화된 배열을 미리 생성해 놓으면 처리 시간을 줄일 수 있습니다.

   배열의 모든 요소가 빈 상태로 배열을 생성할 수 있습니다. 빈 배열은 배열 안에 데이터가 전혀 없는 것이 아니라, 메모리 상에 있는 임의의 값이 할당되어 있습니다. 이후 새로운 데이터가 입력되면 기존 값이 입력된 데이터로 대체됩니다.


빈 배열 생성 함수
np.empty(shape, dtype = np.data type)
  • shape: 배열의 형태를 입력합니다. 2차원 이상의 배열을 생성할 때는 튜플 형태로 입력합니다.
  • data type: 배열의 데이터 유형을 지정합니다. 기본값은 float64입니다.




Code A

   3X2 크기의 2차원 빈 배열을 생성해 봅시다. (단, 데이터 유형은 기본값으로 처리합니다.)

0으로 초기화된 배열 생성하기

   배열의 모든 요소가 0으로 채워진 배열을 생성할 수도 있습니다. 빈 배열과 마찬가지로 데이터를 수집 후 처리할 때 유용하게 사용할 수 있습니다.


0으로 초기화된 배열 생성 함수
np.zeros(shape, dtype = np.data type)
  • shape: 배열의 형태를 입력합니다. 2차원 이상의 배열을 생성할 때는 튜플 형태로 입력합니다.
  • data type: 배열의 데이터 유형을 지정합니다. 기본값은 float64입니다.




Code B

   0으로 초기화된 5X5X2 크기의 3차원 배열을 생성해 봅시다. (단, 데이터 유형은 int32로 처리합니다.)
