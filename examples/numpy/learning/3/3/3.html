넘파이 배열의 기본 연산

기본 수학 함수

넘파이는 배열에 대해 수학적 연산을 수행하는 다양한 함수를 제공합니다. 이를 통해 배열의 합, 평균, 표준편차 등 중요한 통계 값을 계산하거나 삼각함수와 같은 고급 연산을 간단히 수행할 수 있습니다.

기본 수학 함수 - 1 : 배열의 합계, 평균, 최대/최소 값

합계: np.sum

평균: np.mean

최댓값: np.max

최솟값: np.min

표준편차: np.std

분산: np.var

A에 있는 빈칸을 채운 후 A 를 실행하여 아래 결과를 확인하세요

합계: 15
평균: 3.0
최댓값: 5
최솟값: 1
표준편차: 1.4142135623730951
분산: 2.0
정답 보기/숨기기 
          
import numpy as np

# 배열의 기본 통계 함수
arr = np.array([1, 2, 3, 4, 5])
print("합계:", np.sum(arr))
print("평균:", np.mean(arr))
print("최댓값:", np.max(arr))
print("최솟값:", np.min(arr))
print("표준편차:", np.std(arr))
print("분산:", np.var(arr))


기본 수학 함수 - 2 : 삼각함수, 지수 함수, 로그 함수

sin: np.sin

cos: np.cos

지수 함수: np.exp

자연 로그: np.log

정답 보기/숨기기 
          
import numpy as np

# 삼각함수
angles = np.array([0, np.pi / 2, np.pi])
print("sin:", np.sin(angles))
print("cos:", np.cos(angles))

# 지수 함수와 로그 함수
values = np.array([1, 2, 3])
print("지수 함수:", np.exp(values))
print("자연 로그:", np.log(values))
