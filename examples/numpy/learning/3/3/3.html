컴퓨터가 표현하는 난수

   컴퓨터 게임을 하다보면 랜덤 박스에서 아이템을 뽑는 경우가 있습니다. 이때 뽑기 결과를 무작위로 결정되도록 하려면 아무도 예측할 수 없는 난수(ramdom number)를 생성해야 합니다.

   컴퓨터는 일반적으로 자연계에 존재하는 실제 난수를 생성할 수 없습니다. 따라서 수학적 알고리즘을 이용해 ‘의사 난수(pseudo-random number)’를 만들어 냅니다. 의사 난수란 실제 난수처럼 보이게 설계된 수열로, 비록 실제 난수는 아니지만 임의성이 충분히 높아 실제 난수처럼 사용할 수 있습니다.

0과 1 사이의 난수 배열 생성하기

   배열의 모든 요소를 0과 1 사이의 무작위 실수로 채워 배열을 생성할 수 있습니다.


0과 1 사이의 난수 배열 생성 함수
np.random.rand(shape)
  • shape: 배열의 형태를 입력합니다. 2차원 이상의 배열을 생성할 때는 튜플 형태로 입력합니다.

지정 범위의 난수 배열 생성하기

   범위를 지정하여 난수 배열을 생성할 수 있습니다. 지정된 범위 안에 있는 임의의 정수가 입력됩니다.


지정 범위의 난수 배열 생성 함수
np.random.randint(low, high, size = shape)
  • low: 생성할 범위의 시작 숫자입니다.
  • high: 생성할 범위의 마지막 숫자에 1을 더한 값입니다. 10까지 생성하고 싶다면, 11을 입력하면 됩니다.
  • shape: 배열의 형태를 입력합니다. 2차원 이상의 배열을 생성할 때는 튜플 형태로 입력합니다.




Code B

   10부터 30 사이의 난수로 구성된 4 X 4 크기의 2차원 배열을 생성해 봅시다.

표준 정규 분포 난수 배열 생성하기

   자연 현상을 비롯해 현실에서 발생하는 많은 현상들은 대체로 정규 분포를 따릅니다. 따라서 정규 분포를 반영하여 난수 배열을 생성하면 자율 주행 상황을 더 체계적이고 현실적으로 시뮬레이션할 수 있습니다.

   넘파이 전용 함수를 사용해 정규 분포를 따르는 난수를 생성할 수 있습니다. 이때 생성되는 값은 평균이 0이고 표준 편차가 1인 분포, 즉 표준 정규 분포를 반영합니다.


표준 정규 분포 난수 배열 생성 함수
np.random.randn(size_1, size_2, ..., size_n)
  • size_n: 차원별로 배열의 크기를 나열해야 합니다. 다른 난수 배열 생성 함수와 달리 튜플을 사용해 배열의 형태를 지정할 수 없습니다.

난수 재현하기

   난수 생성 함수로 산출된 난수는 실행할 때마다 그 값이 달라집니다. 난수를 생성할 때 사용하는 초기값, 즉 시드(seed)가 매번 달라지기 때문입니다.

   그런데 사용자가 특정 값으로 시드를 고정해 놓으면, 언제든 동일한 난수를 생성할 수 있습니다. 난수 생성을 제어하면 일정한 실험 조건에서 반복적으로 시뮬레이션할 수 있고, 일관된 실험 결과를 얻거나 이전 상황을 재현하는 데 유용합니다.


시드(seed) 설정 함수
np.random.seed(value)
  • value: 난수 생성을 제어하는 초기값입니다. 이후 실행부터 적용되어 동일한 난수 수열이 생성됩니다.

   seed를 50으로 고정한 후 0부터 1 사이의 난수로 구성된 4 크기의 1차원 배열을 세 번 생성해 보고, seed를 40으로 고정한 후 생성한 값과 비교해 봅시다.
