2

데이터 타입 지정하기

   ‘np.array()’는 기본적으로 입력된 반복 가능한 객체의 데이터 유형을 반영하여 넘파이 배열을 생성합니다. 만약 다른 데이터 유형으로 바꾸어 배열을 생성하고 싶다면, dtype 매개변수를 사용하면 됩니다.

   넘파이에서 사용되는 주요 데이터 유형은 정수형, 실수형, 문자형, 불리언형 등이 있습니다. 숫자형은 표현 가능한 숫자에 따라 다양한 비트(bit) 유형으로 다시 구분됩니다.




넘파이 배열 생성 함수(데이터 타입 지정)
np.array(itarable, dtype = np.data type)
  • itarable: 반복 가능한 객체입니다. 보통 리스트나 튜플을 입력합니다.
  • data type: 지정하고자 하는 넘파이 데이터 유형입니다.
