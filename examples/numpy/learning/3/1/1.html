배열의 산술 연산

배열의 산술 연산 -1 : 배열과 스칼라 간 연산

넘파이에서 배열의 각 요소에 스칼라 값을 자동으로 적용하여 연산을 수행하는 기능입니다.

배열의 크기에 상관없이 스칼라 값이 배열 전체로 확장되어 연산됩니다.

A를 출력하여 결과를 확인하세요

          
import numpy as np

# 배열과 스칼라 간 연산
arr = np.array([1, 2, 3, 4])
print("배열:", arr)
print("배열 + 5:", arr + 5)
print("배열 * 2:", arr * 2)


배열의 산술 연산 -2 : 배열 간 덧셈, 뺄셈, 곱셈, 나눗셈

배열 간 산술 연산은 두 배열의 각 요소를 대응시켜 연산합니다.

두 배열의 크기(shape)가 같아야 연산이 가능합니다

넘파이는 배열 간 덧셈(+), 뺄셈(-), 곱셈(*), 나눗셈(/) 등 기본 산술 연산을 간단하게 수행할 수 있도록 지원합니다.

B를 출력하여 결과를 확인하세요

          
import numpy as np

# 두 배열 생성
arr1 = np.array([1, 2, 3, 4])
arr2 = np.array([5, 6, 7, 8])

# 덧셈
addition = arr1 + arr2
print("배열 덧셈:", addition)  # [6, 8, 10, 12]

# 뺄셈
subtraction = arr2 - arr1
print("배열 뺄셈:", subtraction)  # [4, 4, 4, 4]

# 곱셈
multiplication = arr1 * arr2
print("배열 곱셈:", multiplication)  # [5, 12, 21, 32]

# 나눗셈
division = arr2 / arr1
print("배열 나눗셈:", division)  # [5.0, 3.0, 2.333, 2.0]


배열의 산술 연산 -3: 지수 연산

배열의 각 요소에 대해 특정 지수를 적용해 계산합니다.

지수 연산자는 ** 또는 np.power()를 사용합니다.

C를 출력하여 결과를 확인하세요

          
import numpy as np

# 배열 생성
arr = np.array([2, 3, 4, 5])

# 지수 연산
exp_arr = arr ** 2  # 각 요소를 제곱
print("지수 연산 (제곱):", exp_arr)

# np.power() 사용
exp_power = np.power(arr, 3)  # 각 요소를 세제곱
print("지수 연산 (세제곱):", exp_power)


배열의 산술 연산 -4: 나머지 연산

배열의 각 요소에 대해 특정 값을 나눈 나머지를 계산합니다.

나머지 연산자는 % 또는 np.mod()를 사용합니다.

D를 출력하여 결과를 확인하세요

          
import numpy as np

# 나머지 연산
mod_arr = arr % 2  # 각 요소를 2로 나눈 나머지
print("나머지 연산 (modulo):", mod_arr)

# np.mod() 사용
mod_np = np.mod(arr, 3)  # 각 요소를 3으로 나눈 나머지
print("나머지 연산 (np.mod):", mod_np)
