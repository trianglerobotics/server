합계 함수

   합계 함수를 사용해 모든 요소의 합을 계산할 수 있습니다. 기본적으로 배열의 전체 합을 하나의 숫자로 반환하고, 특정 축을 기준으로 합을 구할 수도 있습니다.


합계 함수
np.sum(array, axis = numder, dtype = data type)
  • array: 합계를 구할 배열을 입력합니다.
  • number: 축 번호를 입력하여 합계 기준 축을 지정합니다. 생략 시 전체 배열 요소의 합이 계산됩니다.
  • data type: 계산 결과의 데이터 유형을 지정합니다.

최댓값 함수

   최댓값 함수를 사용해 모든 요소 중 가장 큰 값을 구할 수 있습니다. 기본적으로 전체 요소 중 가장 큰 값을 반환하며, 특정 축을 기준으로 최댓값을 구할 수도 있습니다.

   만약 전체 배열 중에서 어떤 위치에 최댓값이 있는지 알고 싶다면, 최댓값의 인덱스를 반환하는 함수를 사용하면 됩니다.


최댓값 함수

  • number: 축 번호를 입력하여 최댓값을 구할 기준 축을 지정합니다. 생략 시 전체 배열 요소의 합이 계산됩니다.

최댓값 인덱스 반환 함수
np.argmax(array, axis = numder)
  • array: 최댓값의 인덱스를 반환할 배열을 입력합니다.
  • number: 축 번호를 입력하여 최댓값의 인덱스를 구할 기준 축을 지정합니다. 생략 시 전체 배열 요소의 합이 계산됩니다.
