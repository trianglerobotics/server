파이토치가 가져야 할 기능

파이토치를 배우기에 앞서, 딥러닝 계산을 쉽게 해주기 위해선 파이토치에 있어야 하는 기능부터 생각해 보고자 합니다

  • 인공신경망을 만들어 줄 수 있어야 합니다
  • 미니배치 단위의 학습이 가능하도록 데이터를 배치 단위로 넣어 줄 수 있어야 합니다
  • 인공신경망에 데이터를 넣어 주어 계산하는 순전파가 가능해야 합니다
  • 인공신경망의 출력값과 차이의 출력값을 비교하여 손실함수를 계산할 수 있어야 합니다
  • 손실함수를 이용하여 가중치를 업데이트하는 경사하강법 혹은 그레디언트를 계산 ,즉 역전파가 가능해야 합니다

파이토치기능 1: 인공신경망 생성

파이토치에서 인공신경망을 만들기 위해서는 nn 모듈을 가져 와야 합니다 nn은 다음과 같은 인공 신경 망들을 생성 할 수 있습니다

  • 완전연결층 : torch.nn.Linear
  • 합성곱층 : torch.nn.Conv2d
  • 활성화 함수 : torch.nn.ReLU , torch.nn.Sigmoid

pytorch의 nn 모듈을 이용하여 파이토치에서 모델 즉 함수를 정의하고 활용하기 위해서는 파이썬 클래스를 다음과 같은 구조로 이용해야 합니다.

  • 클래스의 init 함수에서는 사용할 인공신경망 모듈을 정의합니다
  • 클래스의 forward 함수에서는 init 함수에서 정의한 함수 모듈들을 복사하여 순전파 구조를 정의합니다.

파이토치기능 3: 순전파

순전파를 하기 위해서는 파이썬의 클래스를 이용하여 모델을 정의한 다음, 클래스의 인스턴스를 만들어 주어야 합니다. “인스턴스.forward(input)”과 같은 형태로 순전파가 가능합니다.

파이토치 순전파
nn.Module.forward(self, *input)
  • self : 인스턴스
  • input : 입력 데이터

파이토치기능 4: 손실함수 계산

순전파를 완료 하였다면 참값와 비교를 위해 손실함수를 통해 손실을 계산하여야 합니다. 파이토치에는 지금까지 학습했던 모든 손실함수가 구현되어 있습니다

  • 평균제곱오차 : torch.nn.MSELoss
  • 교차 엔트로피 : torch.nn.CrossEntropyLoss
  • 이진 교차 엔트로피 : torch.nn.BCELoss

6

파이토치기능 5: 역전파

역전파는 손실함수를 기반으로 모델의 파라미터를 업데이트 하는 과정입니다. 파이토치에서는 손실함수로 나온 결과에 .backward()를 붙여주는 과정을 통해 역전파가 가능합니다

  • 평균제곱 오차 기반 역전파 : nn.MSELoss().backward()
  • 교차 엔트로피 오차기반 역전파 : nn.CrossEntropyLoss().backward()
  • 이진 교차 엔트로피 오차기반 역전파 : nn.BCELoss().backward()
