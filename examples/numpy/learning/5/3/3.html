선형회기 단계

지금까지 학습한 머신러닝의 선형회귀의 개념과 파이토치의 강력한 기능들을 이용하여 선형 회귀를 파이토치로 구현하고 학습까지 진행해 보고자 합니다. 선형 회귀의 단계는 다음과 같았습니다

  • 임의의 직선을 가정하고, 예측할 데이터를 준비합니다
  • x 값을 넣었을 때 예측한 y 값(출력)과 실제 y 값(정답)을 비교 즉 손실 함수를 계산합니다.
  • 손실함수를 줄이기 위해 a,b값을 조금씩 조정합니다
  • 위의 과정을 반복해서 데이터에 가장 잘 맞는 직선을 찾아냅니다
