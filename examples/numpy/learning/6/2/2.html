1차원 배열 슬라이싱(slicing)

   배열 슬라이싱은 배열의 일부를 잘라 내어 특정 구간이나 영역을 선택해 추출하는 기능입니다. 인덱스를 사용해 추출하고자 하는 범위를 선택합니다.




배열 슬라이싱
array[start:end:step]
  • array: 슬라이싱을 진행할 배열입니다.
  • start: 슬라이싱을 시작할 요소의 인덱스입니다. 생략 시 기본값인 0으로 지정됩니다.
  • end: 슬라이싱이 종료되는 지점의 인덱스 번호에 1을 더한 값을 입력합니다. 예를 들어 종료 지점의 인덱스가 3이라면 4를 입력합니다.
  • step: 요소를 선택하는 간격을 입력합니다. 기본값은 1이며, 음수를 입력하면 역방향으로 진행됩니다.




Code A

   슬라이싱을 사용해 array1에 저장된 1차원 배열에서 ‘B’, ‘D’, ‘F’를 한 번에 추출해 봅시다.

다차원 배열 슬라이싱

   축별로 슬라이싱 범위를 지정하면 2차원 이상의 배열도 슬라이싱할 수 있습니다. 특정 차원의 모든 요소를 선택하고 싶다면 값이 없는 빈 콜론(:)을 입력하면 됩니다.




다차원 배열 슬라이싱
array[slicing_0, slicing_1, ..., slicing_n]
  • array: 슬라이싱을 진행할 배열입니다.
  • slicing_n: 축별 슬라이싱 범위입니다. 추출하려는 범위를 축에 따라 ‘[start:end:step]’ 형식으로 순서대로 입력합니다.




Code B

Code B 를 실행하여 출력 결과를 확인하세요.
