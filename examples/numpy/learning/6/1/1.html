넘파이 배열의 인덱스(index)

   넘파이 배열의 각 요소에는 고유한 위치 데이터인 인덱스가 부여됩니다. 정방향 인덱스는 0으로 시작하며, 행(row) 인덱스는 왼쪽부터, 열(column) 인덱스는 상단부터, 깊이(depth) 인덱스는 앞쪽부터 순서대로 부여됩니다.





   역방향 인덱스는 -1부터 시작하며, 행 인덱스는 오른쪽부터, 열 인덱스는 하단부터, 깊이 인덱스는 뒤쪽부터 순서대로 부여됩니다.




2

배열 인덱싱(indexing)

   배열 인덱싱은 사용자가 원하는 요소를 추출하거나 수정할 수 있는 기능입니다. 인덱스를 활용하여 배열 요소의 위치를 지정하고, 해당 위치의 값을 선택하여 추출하거나 변경할 수 있습니다.




배열 인덱싱
array[axis0_idex, axis1_index, ..., axisn_index]
  • array: 인덱싱을 진행할 배열입니다.
  • axisn_index: 추출하려는 요소의 인덱스를 축에 따라 순서대로 입력합니다. 축 번호를 입력하여 합계 기준 축을 지정합니다. 생략 시 전체 배열 요소의 합이 계산됩니다.
