YOLO

YOLO(You Only Look Once)에서 바운딩 박스를 그리는 이유는 객체 검출(Object Detection) 결과를 시각적으로 표현하기 위함입니다. 바운딩 박스는 이미지나 영상에서 검출된 객체의 위치와 크기를 나타내는 사각형 입니다

이미지에 바운딩 박스 그리기

분류할 사물에 마우스를 클릭하여 움직이면 박스가 만들어지게 됩니다


박스가 만들어지게 되면 클래스 선택창이 표시됩니다 박스에 부여할 클래스를 선택한 후 Finish 버튼을 눌러 박스를 저장하세요


이미지별 각각 만들어진 박스, 클래스는 CLASSES 창에서 확인 할 수 있습니다


상단 중앙의 버튼을 눌러 이미지를 교체하여 모든 이미지에 대한 라벨링을 완료하세요

데이터 생성하기

모든 이미지에 대해서 박스를 그렸다면 Generate 버튼을 눌러 데이터를 생성하세요

*데이터 생성 과정은 현재 그린 바운딩 박스를 학습에 필요한 텍스트 파일로 만드는 과정입니다
