1

객체인식 기반 장애물 감지 개요

자율주행 자동차가 장애물을 감지하는 방법중 하나로 인공지능 기반 객체감지를 사용 할 수 있습니다. 장애물로는 15cm*15cm*15cm 검정 스펀지를 사용합니다


2

클래스 설정하기 - 1

감지할 물체에 해당하는 클래스를 만들어야 합니다. Add Classes 버튼을 누르면 다음과 같은 화면이 표시됩니다


3

클래스 설정하기 - 2

입력할 클래스(‘cube’)를 타이핑 한 후 Add 버튼을 누르면 클래스를 생성 할 수 있습니다


4

클래스 관리하기

Modify Classes 를 클릭하면 클래스를 삭제 할 수 있는 창이 표시됩니다.

삭제할 클래스의 체크박스를 클릭 한 후 Execute Job을 클릭하면 클래스를 삭제 할 수 있습니다
