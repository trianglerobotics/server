후방, 좌,우에 있는 장애물 거리 추출하기

라이다에서의 각도는 라이다를 상단에서 바라보았을 때 시계 방향으로 증가합니다. 이에 따라 후방, 좌, 우는 각각 180도,270도, 90도의 값을 가지게 됩니다.

Code B

  Code B 에 후방,좌,우,에 전방에 있는 장애물과의 거리를 추출하는 알고리즘을 작성하세요

angle_index_left = np.argmin(np.abs(angle_array - 270))
angle_index_right = np.argmin(np.abs(angle_array - 90))
angle_index_back = np.argmin(np.abs(angle_array - 180))

distance_left = dist_array[angle_index_left]
distance_right = dist_array[angle_index_right]
