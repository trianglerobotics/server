극좌표계

라이다 데이터는 x,y 좌표가 아니라 각도, 거리가 표시가 됩니다. 이는 극좌표계 (Polar coordinate)로 표시 할 수 있습니다. 극좌표계는 다음과 같은 구성 요소를 가지고 있습니다

  • 극점(Pole) : 좌표계의 원점
  • 극축(Polar Axis) : 기준이 되는 방향
  • r : 극점에서 해당 점까지의 거리(반지름) = 거리
  • θ : 극축을 기준으로 측정한 각도 = 각도

# 목표 지점 (x, y) -> (1, 1)
target_r = np.sqrt(1**2 + 1**2)  # r = 1.41m
target_theta = 45  # 45도

# 가장 가까운 각도 인덱스 찾기
angle_index = np.argmin(np.abs(angle_array - target_theta))

# 거리 확인
if np.abs(dist_array[angle_index] - target_r) <= 0.1:  # 오차 범위 ±0.1m
    print("장애물이 있습니다.")
else:
    print("장애물이 없습니다.")
