파이토치 텐서

넘파이에서 배열을 담기 위해서는 ndarray를 사용하였습니다. 파이토치도 이처럼 자체적인 배열을 담는 기능을 가지고 있습니다. 파이토치에서는 이것을 텐서(Tensor)라고 이름 지었습니다.

  • 넘파이에서는 numpy.ndarray() 으로 배열을 생성합니다
  • 파이토치에서는 torch.tensor() 으로 배열을 생성합니다
파이토치 텐서
torch.tensor(data, dtype=None, device=None, requires_grad=False)
  • data : 배열을 생성할 데이터
  • dtype : 데이터 타입
  • device : 텐서를 생성할 디바이스
  • requires_grad : 텐서에 대한 기울기를 저장할지 여부
