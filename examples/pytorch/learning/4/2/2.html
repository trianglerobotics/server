합성공 신경망의 구성

합성곱 신경망은 다음과 같은 요소들로 구성됩니다

  • 입력층 : 합성곱 신경망에 처음 데이터가 들어오는 계층입니다.
  • 합성곱층 : 합성곱층은 입력 데이터와 필터 를 활용해 특징을 추출합니다
  • 폴링층 : 폴링층은 합성곱층의 출력의 크기를 줄이는 역할을 합니다.
  • 완전연결층 : 모든 뉴런이 서로 연결된 구조로, 데이터를 1차원으로 변환합니다
  • 활성함수 : 합성곱 신경망의 완전 연결층에는 기존과는 다른 활성함수가 사용됩니다.
  • 손실함수 : 분류 문제에 주로 사용되는 합성곱 신경망에는 회귀와는 다른 손실함수가 사용됩니다.
