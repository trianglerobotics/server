가중합

각 노드에서 들어오는 신호에 가중치를 곱한 후에 바이어스를 더해서 활성화 함수로 전달 되는데 이 값들을 모두 더한 것을 가중합 이라고 합니다



A에 있는 빈칸을 채워 아래 결과를 출력하세요

입력값: [2. 3. 5.]
가중치: [0.1 0.2 0.3]
가중합: 2.3
정답 보기/숨기기 
          
import numpy as np

# 입력 데이터 (예: 세 개의 특징값)
inputs = np.array([2.0, 3.0, 5.0])

# 가중치 (weights): 각 입력에 대응되는 가중치 값
weights = np.array([0.1, 0.2, 0.3])

# 가중합 계산: 각 입력값 * 가중치를 모두 더함
# weighted_sum = (입력1 * 가중치1) + (입력2 * 가중치2) + (입력3 * 가중치3)
weighted_sum = np.dot(inputs, weights)

# 결과 출력
print("입력값:", inputs)
print("가중치:", weights)
print("가중합:", weighted_sum)
