바이어스 \( b \)

바이어스는 가중치와 입력값의 곱에 추가적으로 더해지는 값 입니다. 바이어스는 뉴런이 학습 과정에서 특정한 임계값(기준값)을 가지도록 만들어 모델의 성능을 향상시키는 데 기여합니다.



A에 있는 빈칸을 채워 아래 결과를 출력하세요

입력값: [2. 3. 5.]
가중치: [0.1 0.2 0.3]
가중합: 2.3
정답 보기/숨기기 
            
import numpy as np

# 입력 데이터 (예: 두 개의 특징값)
inputs = np.array([1.5, 2.0])

# 가중치 (weights): 각 입력에 대한 가중치 값
weights = np.array([0.4, 0.6])

# 편향 (bias): 모델의 출력을 조정하는 상수
bias = 1.0

# 출력 계산: 가중치와 입력의 합을 구한 뒤 편향을 더함
# z = (입력1 * 가중치1) + (입력2 * 가중치2) + 편향
output = np.dot(inputs, weights) + bias

# 결과 출력
print("입력값:", inputs)
print("가중치:", weights)
print("편향:", bias)
print("출력값 (z):", output)
