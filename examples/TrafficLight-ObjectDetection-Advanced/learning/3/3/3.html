Basic 단계의 한계점

  Basic 단계에서는 신호등 전체를 2개의 클래스로 나누어 감지하도록 했었습니다. 하지만 2개의 클래스의 특징 차이가 크지 않다는 한계점을 가지고 있어 정확도가 높지 않았습니다

  따라서 신호등을 감지할 때 신호등의 색깔이 아닌 신호등 만을 감지 한 뒤에 밝은곳의 위치에 따라 현재 색깔을 판단하고자 합니다

이미지에서 밝기 추출하기

   일반적으로 이미지는 RGB 색공간에서 표현됩니다. 그러나 RGB 색공간은 밝기 정보가 R, G, B 채널에 분산되어 있어, 밝기만을 직접 추출하기가 어렵습니다. 이를 해결하기 위해 RGB 이미지를 HSV 색공간으로 변환하여, 밝기를 나타내는 V(Value) 채널을 추출하는 과정이 필요합니다


OpenCV 색공간 변환 함수
cv2.cvtColor(rgb_image, cv2.COLOR_BGR2HSV)
  • rgb_image : 변환할 이미지
  • cv2.COLOR_BGR2HSV : 변환할 색공간
