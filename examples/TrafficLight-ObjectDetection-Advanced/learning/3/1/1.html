모델 저장의 배경

인공지능 모델은 현재 프로젝트 내부에 저장되게 됩니다. 하지만. 다른 프로젝트 내부에서 인공지능 모델을 사용하려고 하면 모델이 저장된 경로를 수동으로 입력해 주어야 합니다

따라서 자주차 내부 저장소에 새로운 이름으로 모델을 저장하고, 저장된 이름으로 모델을 불러 오려고 합니다


2

모델을 내부메모리에 저장하기 - 1

우선 테스트 과정에서 실행하였던 모델중 저장할 모델의 이름을 클릭합니다


3

모델을 내부메모리에 저장하기 - 2

저장할 모델을 선택했다면 이름을 타이핑 한 후 save 하여 모델을 저장합니다
