신호등 감지 개요

자율주행 자동차가 신호등을 감지할때는 물체감지를 기반으로 신호등의 형태를 감지해야 합니다 Basic 에서는 Red ,Green으 로 클래스를 나누어 신호 감지를 진행, Advanced 에서는 신호등을 감지 후 hsv채널에서 밝기를 감지하는 방식으로 신호를 감지하고자 합니다


2

클래스 설정하기 - 1

감지할 물체에 해당하는 클래스를 만들어야 합니다. Add Classes 버튼을 누르면 다음과 같은 화면이 표시됩니다


3

클래스 설정하기 - 2

입력할 클래스('TrafficLight')를 타이핑 한 후 Add 버튼을 누르면 클래스를 생성 할 수 있습니다


5

클래스 관리하기

Modify Classes 를 클릭하면 클래스를 삭제 할 수 있는 창이 표시됩니다.

삭제할 클래스의 체크박스를 클릭 한 후 Execute Job을 클릭하면 클래스를 삭제 할 수 있습니다
