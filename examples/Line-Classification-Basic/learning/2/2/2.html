학습 파라미터 설정하기

  • EPOCH(에폭) : 전체 데이터로 학습할 횟수

  • TRAIN_DATA_PERCENT(학습데이터 비율) : 전체 데이터에서 학습데이터로 사용할 데이터의 비율
