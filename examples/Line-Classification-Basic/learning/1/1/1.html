1

개요 : 자주차의 동작 클래스 설정하기

자율주행 자동차가 차선을 감지하는 방법으로는 이미지 분류를 사용할 수 있습니다. 간단하게 이미지를 좌회전, 직전 , 우회전으로 나누고, 이를 학습시켜 주행에 활용 할 수 있습니다


2

클래스 설정하기 - 1

차선을 분류하기 위해서 좌회전, 직진 , 우회전에 해당하는 클래스를 만들어 주어야 합니다. Add Classes 버튼을 누르면 다음과 같은 화면이 표시됩니다


3

클래스 설정하기 - 2

입력할 클래스(‘left’)를 타이핑 한 후 Add 버튼을 누르면 클래스를 생성 할 수 있습니다


4

클래스 설정하기 - 3

(‘center’, ’right’ 도 같은 방식으로 반복하여 클래스를 생성합니다)


5

클래스 관리하기

Modify Classes 를 클릭하면 클래스를 삭제 할 수 있는 창이 표시됩니다.

삭제할 클래스의 체크박스를 클릭 한 후 Execute Job을 클릭하면 클래스를 삭제 할 수 있습니다
