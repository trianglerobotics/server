장애물 거리 도출하기

물리적 준비사항

STEP-1 : 자주차 앞에 차보다 큰 장애물을 설치합니다



CELL-1

STEP-1 : CELL-1 을 실행하여 자주차 라이브러리를 가져옵니다



CELL-2

STEP-1 : CELL-2를 실행하기전 이미지 뷰어를 열어준다음 CELL-2 코드를 실행합니다

STEP-2 : 장애물의 거리 이미지가 나온것을 확인 합니다



CELL-3

STEP-1 : CELL-3 을 실행합니다 , CELL-3 은 중간의 60x60 크기의 사각형의 평균을 구하여 거리를 구하는 예제 입니다

STEP-2 : 평균값이 출력된것을 확인 합니다

TIP* : 출력된 값은 밝기 값으로 0-255 값을 가지고 있습니다 값이 크다는것은 밝다는것을 의미하고, 거리가 가까울수록 값이 증가합니다
