자주차 모듈 가져오기

자주차를 제어하기 위한 함수들의 모음집인 jajucha2 라이브러리를 가져옵니다

A

자주차 라이브러리는 인공지능 기능을 포함 하고 있어 로딩하는데 2-3초 정도 소요됩니다

import jajucha2


주행하기

그리드 기반의 알고리즘으로 자주차를 주행합니다

B

while True: image = jajucha2.camera.get_image() (V, L, R), grid = jajucha2.camera.gridFront(image) jajucha2.camera.show_image(grid) if(V[5] < 100): jajucha2.control.set_motor(-10,-10,5) else: jajucha2.control.set_motor(0,0,5)


while True: -> 무한 반복합니다

image = jajucha2.camera.get_image() -> 중간 카메라에서 이미지를 가져옵니다

(V, L, R), grid = jajucha2.camera.gridFront(image) -> 이미지에 그리드를 그리고 V,L,R 그리드 이미지를 가져옵니다

if(V[5] < 100): -> 만약 V[5]의 값이 100보다 작다면

jajucha2.control.set_motor(-10,-10,5) -> 조향각을 왼쪽으로 조정합니다

else: -> if를 충족하지 않는다면

jajucha2.control.set_motor(0,0,5) -> 자주차를 직진합니다
