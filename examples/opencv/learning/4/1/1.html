이미지 크기 조정하기

   자율주행자동차는 다양한 센서로 얻은 정보를 실시간으로 처리하기 때문에, 고해상도인 원본 이미지를 그대로 사용하면 처리 속도가 느려질 수 있습니다. 따라서 이미지를 적절한 크기로 조정하여 자율주행 시스템의 효율성을 높이고 성능을 향상시켜야 합니다.


이미지 크기 조정 함수
cv2.resize(data, dsize, fx=0, fy=0, interpolation=flag)
  • data: 크기를 조정할 이미지 데이터입니다.
  • dsize: 조정할 이미지의 크기입니다. ‘(400, 300)’처럼 너비와 높이 순으로 입력하면 됩니다.
  • fx/fy: ‘fx'와 ’fy'는 가로 방향과 세로 방향의 비율입니다. 이미지를 절반 크기로 줄이고 싶다면 ‘fx=0.5, fy=0.5’로 입력하면 됩니다. size와 동시에 사용할 수는 없습니다.
  • interpolation: 크기 조정 시 사용할 보간법을 입력합니다.

2

이미지 보간법

   이미지의 확대할 때 생기는 빈 공간을 자연스럽게 처리하거나, 이미지를 축소할 때 여러 픽셀을 하나의 픽셀로 압축하는 방법을 수학적 용어로 보간법이라고 합니다.

   다양한 보간법을 사용하여 이미지 크기를 조정하는 과정에서 주변 픽셀의 색상을 참고해 픽셀에 입력될 값을 결정할 수 있습니다.


보간법(interpolation) flags
① cv2.INTER_LINEAR: 주변에 있는 픽셀 2개의 값을 가중치 기반으로 보간하여 픽셀 값을 결정합니다. 기본값으로 설정되어 있습니다.
② cv2.INTER_AREA: 축소에 최적화된 보간법입니다. 축소된 영역에 속한 픽셀 값을 평균하여 새로운 값을 도출합니다.
③ cv2.INTER_CUBIC: 확대에 최적화된 보간법입니다. 주변 4x4 영역에 있는 픽셀 값을 가중치 기반으로 보간하여 픽셀의 값을 결정합니다.
④ cv2.INTER_NEAREST: 속도가 가장 빠른 보간법입니다. 가장 가까운 픽셀 값을 그대로 사용합니다. 계단 현상 같은 품질 저하가 나타날 수 있습니다.
⑤ cv2.INTER_LANCZOS4: 가장 높은 품질을 제공하는 보간법입니다. 주변 8x8 영역에 있는 픽셀 값을 사인(sine) 함수로 보간하여 픽셀의 값을 결정합니다.
