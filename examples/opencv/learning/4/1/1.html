이미지 블러링

OpenCV에서 **이미지 블러링(Blurring)**은 노이즈 제거나 부드럽게 만드는 작업에 사용됩니다. 블러링은 주로 이미지를 전처리하거나 배경을 흐리게 만들 때 활용됩니다. OpenCV는 다양한 블러링 기법을 제공합니다.

이미지 블러링 -1 :평균 블러링 (Averaging)

이미지의 각 픽셀 값을 주변 픽셀의 평균으로 대체합니다.

cv2.blur(src, ksize)

src: 입력 이미지 (numpy.ndarray 형식).

ksize: 커널 크기 (width, height)

A 를 실행하기 전에 를 클릭하여 이미지 뷰어를 활성화 한 후에 결과를 확인하세요 

          
import cv2
import jajucha2
image = cv2.imread('example.jpg')

# 평균 블러링
blurred_image = cv2.blur(image, (5, 5))

jajucha2.camera.show_image(blurred_image)


이미지 블러링 -2 : 가우시안 블러링 (Gaussian Blurring)

가우시안 커널을 사용하여 블러링을 수행하며, 부드러운 블러링 효과를 제공합니다.

cv2.GaussianBlur(src, ksize, sigmaX)

src: 입력 이미지 (numpy.ndarray 형식).

ksize: 커널 크기 (width, height) (홀수).

sigmaX: X축 방향의 가우시안 표준편차

B 를 실행하기 전에 를 클릭하여 이미지 뷰어를 활성화 한 후에 결과를 확인하세요 

                  
import cv2
import jajucha2

image = cv2.imread('example.jpg')

# 가우시안 블러링
gaussian_blurred = cv2.GaussianBlur(image, (5, 5), 0)

jajucha2.camera.show_image(gaussian_blurred)


이미지 블러링 -3 : 미디언 블러링 (Median Blurring)

커널 내 픽셀 값의 중간값으로 대체합니다. 특히, 소금-후추 노이즈(Salt-and-Pepper Noise)에 효과적입니다.

cv2.medianBlur(src, ksize)

src: 입력 이미지 (numpy.ndarray 형식).

ksize: 커널 크기 (홀수).

C 를 실행하기 전에 를 클릭하여 이미지 뷰어를 활성화 한 후에 결과를 확인하세요 

                  
import cv2
import jajucha2

image = cv2.imread('example.jpg')

# 미디언 블러링
median_blurred = cv2.medianBlur(image, 5)

jajucha2.camera.show_image(median_blurred)
