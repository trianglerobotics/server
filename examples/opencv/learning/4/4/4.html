1

그래픽을 삽입하는 이유

  실행 결과를 이미지 위에 그래픽으로 표현하면, 설계한 코드가 제대로 작동하는지 확인할 수 있습니다. 이를 통해 알고리즘이 정상적으로 작동하는지 확인하고, 오류가 있다면 수정하는 디버깅(debugging)을 수행할 수 있습니다.



  그래픽을 삽입하는 OpenCV 전용 함수를 사용하면 이미지에 선, 도형, 텍스트 같은 요소를 쉽게 표현할 수 있습니다.



2

직선 그리기


직선 생성 함수
cv2.line(data, start, end, color, thickness=None)
  • data: 그래픽을 삽입할 이미지 데이터입니다.
  • start: 직선의 시작 좌표입니다. 튜플 형식으로 작성하면 됩니다.
  • end: 직선의 끝 좌표입니다.
  • color: 직선의 색상입니다. BGR 순서에 맞추어 0부터 255 사이의 값을 ‘(value1, value2, value3)’ 형식으로 작성하면 됩니다.
  • thickess: 직선의 두께를 의미하며 기본값은 1입니다. 픽셀 단위로 두께를 설정하기 때문에 정수만 입력할 수 있습니다.




Code A

   직선 생성 함수를 사용해 1024 X 1024 크기인 예제 이미지에 아래 사진과 같이 두께가 10인 빨간색 직선을 삽입해 봅시다.



Code A 를 실행하여 출력 결과를 확인하세요.


3

사각형 그리기


사각형 생성 함수
cv2.rectangle(data, start, end, color, thickness=None)
  • data: 그래픽을 삽입할 이미지 데이터입니다.
  • start: 왼쪽 상단 꼭짓점 좌표입니다.
  • end: 오른쪽 하단 꼭짓점 좌표입니다.
  • color: 테두리의 색상입니다. BGR 순서에 맞추어 0부터 255 사이의 값을 ‘(value1, value2, value3)’ 형식으로 작성하면 됩니다.
  • thickess: 테두리 두께를 의미하며 기본값은 1입니다. -1을 입력하면 도형 내부에 색상이 채워집니다.




Code B

   사각형 생성 함수를 사용해 1024 X 1024 크기인 예제 이미지에 아래 사진과 같이 내부가 채워진 초록색 사각형을 삽입해 봅시다.



Code B 를 실행하여 출력 결과를 확인하세요.


4

원 그리기


원 생성 함수
cv2.circle(data, center, radius, color, thickness=None)
  • data: 그래픽을 삽입할 이미지 데이터입니다.
  • center: 원의 중심 좌표입니다.
  • radius: 원의 반지름입니다. 픽셀 단위로 길이를 표현하기 때문에 정수만 입력할 수 있습니다.
  • color: 테두리의 색상입니다. BGR 순서에 맞추어 0부터 255 사이의 값을 ‘(value1, value2, value3)’ 형식으로 작성하면 됩니다.
  • thickess: 테두리 두께를 의미하며 기본값은 1입니다. -1을 입력하면 도형 내부에 색상이 채워집니다.




Code C

   원 생성 함수를 사용해 1024 X 1024 크기인 예제 이미지에 아래 사진과 같이 두께가 4인 파란색 원을 삽입해 봅시다.

5

텍스트 입력하기


텍스트 입력 함수
cv2.putText(data, text, point, fontFace, fontSize, color, thickness=None)
  • data: 그래픽을 삽입할 이미지 데이터입니다.
  • text: 입력할 문자열입니다.
  • point: 문자열을 삽입할 좌표입니다. 입력한 좌표에서부터 우측 방향으로 텍스트가 입력됩니다.
  • fontFace: 폰트 유형, 쉽게 말해 글씨체입니다. 원하는 폰트 flag를 입력하면 됩니다.
  • fontSize: 폰트 크기이며 기본값은 1입니다. 1을 기준으로 크기가 달라지며 소수점 단위로 입력할 수 있습니다. 반드시 해상도에 비례하지 않으며 입력된 숫자에 따라 상대적인 크기로 지정됩니다.
  • color: 테두리의 색상입니다. BGR 순서에 맞추어 0부터 255 사이의 값을 ‘(value1, value2, value3)’ 형식으로 작성하면 됩니다.
  • thickness: 폰트 두께를 의미하며 기본값은 1입니다.


폰트 유형 flags
② cv2.FONT_HERSHEY_SIMPLEX: 기본값으로 설정된 폰트입니다. 일반적인 산세리프 폰트입니다.
② cv2.FONT_HERSHEY_PLAIN: 얇은 산세리프 폰트입니다.
③ cv2.FONT_HERSHEY_DUPLEX: 굵은 산세리프 폰트입니다.




Code D
