OPENCV 기본 사용법

이미지 저장하기

OpenCV에서 이미지를 저장하려면 cv2.imwrite() 함수를 사용합니다. 이 함수는 이미지를 디스크에 저장하며, 다양한 파일 포맷(.jpg, .png, .bmp, 등)을 지원합니다.

cv2.imwrite(filename, image, params=None)

filename: 저장할 파일의 경로

image: 저장할 이미지

params: 특정 파일 포맷의 저장 품질이나 설정

     cv2.IMWRITE_JPEG_QUALITY JPEG 품질

     cv2.IMWRITE_PNG_COMPRESSIONPNG 압축 레벨

A 를 실행하여 결과를 확인하세요 를 클릭하면 저장된 이미지를 확인할 수 있습니다

          
import cv2

# 이미지 읽기
image = cv2.imread('example.jpg')

# 이미지 저장
cv2.imwrite('output.jpg', image)  # JPG 형식으로 저장
print("이미지가 저장되었습니다.")
