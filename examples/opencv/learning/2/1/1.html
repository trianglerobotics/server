OPENCV 기본 사용법

이미지 읽기

OpenCV에서 이미지를 읽을 때 주로 사용하는 함수는 cv2.imread()입니다. 이 함수는 디스크에서 이미지를 로드하여 OpenCV가 처리할 수 있는 형태(numpy.ndarray)로 반환합니다.

cv2.imread(filename, flags)

filename: 이미지 파일 경로

flags: 이미지 읽기 방식

     cv2.IMREAD_COLOR (기본값) BGR 컬러로 읽음

     cv2.IMREAD_GRAYSCALE Grayscale(흑백)로 읽음

A 를 실행하여 결과를 확인하세요

        
import cv2

# 컬러 이미지 읽기 (기본값)
image = cv2.imread('example.jpg', cv2.IMREAD_COLOR)
print("이미지 데이터 타입:", type(image))  # numpy.ndarray
print("이미지 크기:", image.shape)  # (높이, 너비, 채널 수)
