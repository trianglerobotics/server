이미지 처리 기초

이미지 크기 조정

OpenCV에서 이미지를 크기 조정(Resizing)하려면 cv2.resize() 함수를 사용합니다. 이 함수는 이미지의 너비와 높이를 조정하며, 다양한 보간법(Interpolation)을 통해 이미지 품질을 관리합니다.

cv2.resize(src, dsize, dst=None, fx=0, fy=0, interpolation=cv2.INTER_LINEAR)

src: 입력 이미지

dsize: 출력 이미지 크기를 지정하는 튜플 (width, height)

dst: 출력 이미지의 저장 위치

fx: 가로 방향의 크기 배율

fy: 세로 방향의 크기 배율

interpolation : 크기를 조정할 때 사용할 보간법

     cv2.INTER_NEAREST 최근접 이웃 보간법

     cv2.cv2.INTER_LINEAR선형 보간법 (기본값)

     cv2.INTER_CUBIC3차 보간법

     cv2.INTER_AREA영역 보간법

     cv2.INTER_LANCZOS4Lanczos 보간법

A 를 실행하여 결과를 확인하세요

        
import cv2

image = cv2.imread('example.jpg')
resized = cv2.resize(image, (200, 100))  # (width, height)
print("새 이미지 크기:", resized.shape)

image = cv2.imread('example.jpg')
resized = cv2.resize(image, None, fx=0.5, fy=0.5, interpolation=cv2.INTER_LINEAR)
print("새 이미지 크기:", resized.shape)
