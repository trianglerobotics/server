이미지 처리 기초

색상변환

OpenCV에서 색상 변환은 cv2.cvtColor() 함수를 사용하여 다양한 색상 공간 간에 이미지를 변환하는 작업입니다. OpenCV는 여러 색상 공간(BGR, RGB, Grayscale, HSV 등)을 지원하며, 이를 통해 이미지 처리를 효율적으로 수행할 수 있습니다.

cv2.cvtColor(src, code)

src: 입력 이미지 (numpy.ndarray 형식).

code: 색상 변환 코드를 지정.

     cv2.COLOR_BGR2RGB BGR에서 RGB로 변환

     cv2.COLOR_RGB2BGR RGB에서 BGR로 변환

     cv2.COLOR_BGR2GRAY BGR에서 Grayscale로 변환

     cv2.COLOR_GRAY2BGR Grayscale에서 BGR로 변환

     cv2.COLOR_BGR2HSV BGR에서 HSV로 변환

     cv2.COLOR_HSV2BGR HSV에서 BGR로 변환

A 를 실행하기 전에 를 클릭하여 이미지 뷰어를 활성화 한 후에 결과를 확인하세요 

          
import cv2
import jajucha2
# 이미지 읽기
image = cv2.imread('example.jpg')

# BGR → RGB 변환
gray_image = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)

# 결과 보기
jajucha2.camera.show_image(gray_image)
