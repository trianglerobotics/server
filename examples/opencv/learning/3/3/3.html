이미지 저장하기

   컴퓨터에 이미지를 저장하기 위해서는 픽셀 데이터를 파일로 변환해서 하드디스크에 기록해야 합니다. OpenCV 전용 함수로 이미지를 간편하게 저장할 수 있습니다.


이미지 저장하기 함수
cv2.imwrite(file_path, data, prameters)
  • file_path: 저장할 이미지 파일의 경로와 이름입니다. "example.jpg"처럼 파일명 뒤에 확장자를 지정하면 해당 형식으로 파일이 저장됩니다.
  • data: 저장할 이미지 데이터입니다. OpenCV에서 작업한 이미지를 입력하면 됩니다.
  • parameters: 매개 변수를 의미하며 함수가 수행하는 작업의 세부적인 설정을 조정합니다. ‘cv2.imwrite()’에서는 저장되는 데이터의 품질이나 압축 수준을 결정합니다.

이미지 저장하기 매개 변수 prameters
① [cv2.IMWRITE_JPEG_QUALITY, value]: JPEG로 저장합니다. 0부터 100까지 이미지 품질을 결정하여 value에 입력할 수 있습니다.
② [cv2.IMWRITE_PNG_COMPRESSION, value]:PNG로 저장합니다. 0부터 9까지 압축 수준을 결정하여 value에 입력할 수 있습니다.

이미지 저장 형식

   이미지를 저장하는 대표적인 형식에는 JPEG, PNG, BMP가 있습니다. 사용하고자 하는 이미지 형식의 확장자를 입력하여 저장할 수 있습니다.

   JPEG는 파일 크기가 작아 많은 사람들이 사용합니다. 하지만 그만큼 손실이 많아 원본 품질을 유지하지 못하는 경우가 많습니다.

   PNG는 파일 크기를 압축할 수 있으면서 원본 품질을 유지합니다. 투명도까지 지원하여 그래픽 디자인에 자주 사용됩니다.

   BMP는 무압축 방식으로 고품질 이미지를 저장합니다. 파일 크기가 커서 일반적인 용도로는 잘 사용되지 않습니다.

특징 JPEG PNG BMP
압축 방식 손실 압축 비손실 압축 압축 없음
파일 크기 작음 큼 중간
품질 압축률에 따라 상이 원본 품질 원본 품질
알파 채널 지원 안 함 지원 지원 안 함
