이미지 처리 기초

이미지 자르기

OpenCV에서 이미지를 자르기(Cropping) 위해 별도의 함수는 필요하지 않습니다. 이미지를 나타내는 배열(numpy 배열)의 슬라이싱(Slicing) 기능을 사용하여 원하는 영역을 추출하면 됩니다.

cropped_image = image[y_start:y_end, x_start:x_end]

y_start : y_end -> 이미지의 세로(행) 범위

x_start : x_end -> 이미지의 가로(열) 범위

A 를 실행하기 전에 를 클릭하여 이미지 뷰어를 활성화 한 후에 A에 있는 빈칸을 채운 후 코드를 실행하세요

정답 보기/숨기기 
        
import cv2
import jajucha2

# 이미지 읽기
image = cv2.imread('example.jpg')

# 자를 범위 지정 (y, x 순서)
x_start, x_end = 100, 400  # 가로 범위
y_start, y_end = 50, 300   # 세로 범위

# 이미지 자르기
cropped_image = image[y_start:y_end, x_start:x_end]

# 결과 보기
jajucha2.camera.show_image(image,'center')
jajucha2.camera.show_image(cropped_image,'left')
