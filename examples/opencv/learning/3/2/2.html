이미지 확인하기

  이미지를 불러와 작업했다면 그 결과를 출력하여 확인해야 합니다. 일반적으로 OpenCV 전용 함수를 사용해 이미지를 출력하지만, 자주차 소프트웨어에서는 자주차 전용 함수로 이미지를 출력합니다.


자주차 SW 이미지 확인 함수
jajucha2.camera.show_image(data, window_name, quality =None)
  • data: 확인할 이미지 파일입니다. 넘파이 배열로 불러온 파일을 입력하면 됩니다.
  • window_name: 이미지 파일을 띄울 창의 이름입니다. "center", "left", "right", "depth", "lidar" 중 하나를 입력하면 됩니다. 이미지 창은 우측 상단의 카메라 아이콘을 누르면 확인할 수 있습니다.
  • quality: 전송할 이미지의 품질입니다. 기본값은 80이며 0부터 100까지 입력할 수 있습니다. 값이 높아질수록 품질은 향상되지만 전송 속도가 낮아집니다.




Code A

   예제 이미지를 컬러로 불러오고, 자주차 전용 이미지 확인 함수를 사용해 이미지를 출력해 봅시다.

Code A 를 실행하여 출력 결과를 확인하세요.




Code B

   예제 이미지를 흑백으로 불러오고, 자주차 전용 이미지 확인 함수를 사용해 이미지를 출력해 봅시다.

Code B 를 실행하여 출력 결과를 확인하세요.




Code C

   예제 이미지를 알파 채널을 포함하는 원본 이미지로 불러오고, 자주차 전용 이미지 확인 함수를 사용해 이미지를 출력해 봅시다.

Code C 를 실행하여 출력 결과를 확인하세요.
