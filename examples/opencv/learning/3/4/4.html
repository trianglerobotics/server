이미지 처리 기초

기본 그리기

OpenCV에서는 기본적인 도형(선, 사각형, 원, 다각형 등)을 그릴 수 있는 함수들이 제공됩니다. 이미지를 생성하거나 기존 이미지 위에 다양한 도형을 추가할 때 사용됩니다.

기본 그리기-1 : 선 그리기

cv2.line(image, pt1, pt2, color, thickness, lineType)

pt1, pt2: 선의 시작점과 끝점 (x, y)

color: 선의 색상 (BGR 값, 예: (255, 0, 0)은 파란색).

thickness: 선의 두께 (기본값: 1)

lineType: 선의 유형 (cv2.LINE_8, cv2.LINE_AA 등)



기본 그리기-2 : 사각형 그리기

cv2.rectangle(image, pt1, pt2, color, thickness)

pt1, pt2: 사각형의 왼쪽 위와 오른쪽 아래 모서리 좌표.

thickness: 선 두께 (음수 값은 내부를 채움)



기본 그리기-3 : 원 그리기

cv2.circle(image, center, radius, color, thickness)

center: 원의 중심 좌표 (x, y)

radius: 반지름

thickness: 선 두께 (음수 값은 내부를 채움)



기본 그리기-4 : 다각형 그리기

cv2.polylines(image, pts, isClosed, color, thickness)

pts: 꼭짓점의 배열 (NumPy 배열로 전달)

isClosed: 닫힌 다각형인지 여부 (True 또는 False).



기본 그리기-5 : 텍스트 추가

cv2.putText(image, text, org, fontFace, fontScale, color, thickness, lineType)

text: 출력할 문자열

org: 텍스트 시작 위치 (x, y)

fontFace: 글꼴 (예: cv2.FONT_HERSHEY_SIMPLEX)

fontScale: 글꼴 크기

lineType: 선 유형 (cv2.LINE_AA 추천)

A에 있는 빈칸을 채운 후 A 를 실행하기 전에 를 클릭하여 이미지 뷰어를 활성화 한 후에 코드를 실행하세요

정답 보기/숨기기 
        
import cv2
import numpy as np
import jajucha2

# 빈 이미지 생성 (검정 배경)
image = np.zeros((500, 500, 3), dtype=np.uint8)

# 선 그리기
cv2.line(image, (50, 50), (450, 50), (255, 0, 0), 5)

# 사각형 그리기
cv2.rectangle(image, (50, 100), (200, 300), (0, 255, 0), 3)

# 원 그리기
cv2.circle(image, (300, 200), 50, (0, 0, 255), -1)

# 다각형 그리기
points = np.array([[100, 400], [200, 350], [300, 400], [200, 450]], np.int32)
cv2.polylines(image, [points], isClosed=True, color=(255, 0, 255), thickness=3)

# 텍스트 추가
cv2.putText(image, "OpenCV Drawing", (50, 450), cv2.FONT_HERSHEY_SIMPLEX, 1, (255, 255, 255), 2)

# 결과 보기
jajucha2.camera.show_image(image)
