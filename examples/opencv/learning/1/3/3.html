OPENCV 기본개념

픽셀의 개념

**픽셀(Pixel)**은 디지털 이미지를 구성하는 가장 작은 단위로, **"picture element"**의 줄임말입니다. 픽셀은 디지털 이미지를 나타내는 기본 요소로, 각 픽셀은 특정 색상 또는 밝기 값을 가지고 있습니다. 컴퓨터 화면, 스마트폰, 디지털 카메라에서 보여지는 모든 디지털 이미지는 이 픽셀들의 집합으로 표현됩니다.

픽셀의 개념 -1 : 컬러 픽셀

각 픽셀은 여러 값(일반적으로 RGB 채널의 값, OPENCV 에서는 BGR 순서)을 가지며, 각 값은 색상 강도를 나타냅니다.

8비트(PAL8), OPENCV 색공간을 기준으로 (255, 0, 0)은 파란색, (0, 255, 0)은 초록색, (0, 0, 255)는 빨간색을 의미 합니다

A 를 실행하여 결과를 확인하세요

        
import cv2

# 이미지 읽기
image = cv2.imread('example.jpg')  # 컬러 이미지 로드

# 특정 픽셀의 BGR 값 가져오기
x, y = 100, 50  # (x, y) 위치
(b, g, r) = image[y, x]  # OpenCV는 (y, x) 순서로 접근
print(f"픽셀 값 (B, G, R): {b}, {g}, {r}")


픽셀의 개념 -2 : 그레이스케일 픽셀

그레이스케일 이미지는 흑백 이미지로, 각 픽셀은 0~255 사이의 값으로 표현됩니다. 0은 검은색, 255는 흰색을 의미하며, 중간 값은 회색을 나타냅니다.

B 를 실행하여 결과를 확인하세요

        
import cv2

# 이미지 읽기 (컬러 이미지 로드)
image = cv2.imread('example.jpg')

# Grayscale로 변환
gray_image = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)

# 특정 픽셀의 Grayscale 값 가져오기
x, y = 100, 50  # (x, y) 위치
gray_value = gray_image[y, x]  # Grayscale 이미지는 단일 채널이므로 하나의 값만 반환
print(f"픽셀의 Grayscale 값: {gray_value}")
