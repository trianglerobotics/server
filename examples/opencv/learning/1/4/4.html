OPENCV 기본개념

색상 공간

OpenCV는 다양한 색상 공간(Color Spaces)을 다루며, 이미지를 서로 다른 색상 공간으로 변환할 수 있도록 여러 변환 함수를 제공합니다. 아래는 OpenCV에서 자주 사용하는 색상 공간과 그 활용에 대한 설명입니다.

색상 공간 -1 : BGR (Blue, Green, Red)

OpenCV에서 기본 색상 공간으로 사용됩니다.

이미지를 cv2.imread()로 로드하면 numpy.ndarray형태의 BGR 형식으로 읽힙니다.

A 를 실행하여 결과를 확인하세요

        
import cv2

# 이미지 읽기
image = cv2.imread('example.jpg')  # 컬러 이미지 로드

# 특정 픽셀의 BGR 값 가져오기
x, y = 100, 50  # (x, y) 위치
(b, g, r) = image[y, x]  # OpenCV는 (y, x) 순서로 접근
print(f"픽셀 값 (B, G, R): {b}, {g}, {r}")
print("이미지 데이터 타입:", type(image))


색상 공간 -2 : HSV (Hue, Saturation, Value)

색상(Hue), 채도(Saturation), 명도(Value)로 구성된 색상 공간입니다.

Hue (색상): 0~179 (OpenCV 기준)

Saturation (채도): 0~255

Value (명도): 0~255

B 를 실행하여 결과를 확인하세요

        
import cv2

# 이미지 읽기
image = cv2.imread('example.jpg')  # 컬러 이미지 로드

# BGR → HSV 변환
hsv_image = cv2.cvtColor(image, cv2.COLOR_BGR2HSV)

# 특정 픽셀의 HSV 값 가져오기
x, y = 100, 50  # (x, y) 위치
(h, s, v) = hsv_image[y, x]  # OpenCV는 (y, x) 순서로 접근
print(f"픽셀 값 (H, S, V): {h}, {s}, {v}")
print("이미지 데이터 타입:", type(image))
